# 량타쯔 (両塔子 / ryou tatsu)

| 항목 | 내용 |
|------|------|
| 일본어 | 両塔子 |
| 로마자 | ryou tatsu |
| 한국어 | 량타쯔 |

## 정의

연속된 2개의 수패. 양옆으로 연속된 수패가 한 장 더 붙으면 슌쯔가 된다. 2장으로 최대 2종 8패를 기다리는 형태이므로 패효율적으로 우수한 형태이다. e.g.) 34만, 67통.
