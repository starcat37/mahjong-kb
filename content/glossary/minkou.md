# 밍커 (明刻 / minkou)

| 항목 | 내용 |
|------|------|
| 일본어 | 明刻 (ミンコ) |
| 로마자 | minkou |
| 한국어 | 밍커 |

## 정의

밍커(明刻)는 다른 플레이어의 타패를 퐁(ポン)하여 만든 커쯔이다.

## 상세 설명

밍커는 안커보다 부수가 절반이다(중장패 밍커 4부, 요구패 밍커 8부). 론으로 완성한 커쯔도 밍커로 취급한다. 밍커를 만들면 멘젠 상태가 깨져 리치를 걸 수 없게 된다.

## 사용 예시

- "퐁으로 발 밍커를 만들어 역패를 확정했다."
- "밍커는 안커보다 부수가 낮다."
