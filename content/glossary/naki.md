# 울기 (鳴き / naki)

| 항목 | 내용 |
|------|------|
| 일본어 | 鳴き (なき) |
| 로마자 | naki |
| 한국어 | 울기 |

## 정의
울기(鳴き/나키)는 다른 플레이어가 버린 패를 가져와 멘쯔를 만드는 행위의 총칭.

## 상세 설명
치(チー)·퐁(ポン)·밍깡(明槓)이 울기에 해당한다. 울기를 하면 멘젠이 깨져 리치를 걸 수 없고, 일부 역이 성립하지 않거나 한수 내려간다(쿠이사가리). 빠른 화료를 위해 전략적으로 활용한다.

## 사용 예시
- "울기를 하면 빠르지만 점수가 낮아질 수 있다"
- "멘젠 유지가 유리한지 울기가 유리한지 판단이 중요"
