# 소메테 (染め手 / somete)

| 항목 | 내용 |
|------|------|
| 일본어 | 染め手 |
| 로마자 | somete |
| 한국어 | 소메테 |

## 정의

혼일색, 혹은 청일색처럼 한 종류의 수패로만 손을 채워나가는 것, 혹은 그렇게 예상되는 손패를 뜻하는 말. 직역하면 '물들이고 있는 손'.
