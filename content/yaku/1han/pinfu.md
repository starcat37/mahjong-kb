# 핑후 (平和)

| 항목 | 내용 |
|------|------|
| 일본어 | 平和 |
| 로마자 | pinfu |
| 한국어 | 핑후 |
| 판수 | 1판 |
| 후로 가능 | 불가 |
| 쿠이사가리 | 없음 |

## 개요

핑후는 멘젠 상태에서 4개의 슌쯔와 역패가 아닌 머리로 구성하며, 반드시 양면대기로 텐파이해야 성립하는 역이다. 부수(符)가 최소화되는 형태로, 리치마작에서 가장 기본적이면서도 빈번하게 등장하는 역이다.

## 성립 조건

1. 멘젠 상태여야 한다.
2. 4개의 멘쯔가 모두 슌쯔(순서대로 이어지는 3장)여야 한다.
3. 머리가 역패(삼원패, 장풍패, 자풍패)가 아니어야 한다.
4. 대기 형태가 양면대기여야 한다 (칸짱, 펜짱, 샤봉, 단기 대기 불가).

## 예시

### 기본 예시

만수 123, 통수 456, 삭수 345, 삭수 678, 만수 55 — 4개 슌쯔 + 역패 아닌 머리(만수 5), 양면대기로 핑후 성립.

### 주의할 패턴

- 머리가 백, 발, 중(삼원패 3종)이면 핑후가 불성립한다.
- 머리가 자풍패이면 불성립한다. 예: 자신이 동가이면 동을 머리로 사용 불가.
- 머리가 장풍패이면 불성립한다. 예: 현재 동X국이면 동을 머리로 사용 불가.
- **자풍도 장풍도 아닌 객풍패는 머리로 사용 가능**하다. 예: 남2국에서 북가라면 서를 머리로 써도 핑후 성립.
- 치를 받는 등 후로를 하면 멘젠이 깨지므로 성립하지 않는다.
- 만수 12 대기(펜짱)나 만수 35 대기(칸짱)처럼 양면이 아닌 대기는 불성립이다.
- 커쯔(같은 패 3장)나 깡쯔가 하나라도 있으면 성립하지 않는다.

## 전략적 활용

- 탕야오와 가장 잘 어울리는 복합 역이다. 핑후 + 탕야오는 멘젠 2판의 기본 조합이다.
- 리치와 함께 선언하면 최소 2판 + 우라도라 기대값까지 올라간다.
- 슌쯔 중심으로 손패를 구성하는 연습은 효율적인 타패 판단력을 기르는 데 도움이 된다.
- 초보자는 핑후 형태를 먼저 익히면 손패 효율을 자연스럽게 이해할 수 있다.
