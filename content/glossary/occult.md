# 오컬트 (オカルト / okaruto)

| 항목 | 내용 |
|------|------|
| 일본어 | オカルト |
| 로마자 | okaruto |
| 한국어 | 오컬트 |

## 정의

마작에서 기세나 흐름 등 눈으로 확인하거나 증명할 수 없는 무언가가 있음을 인정하고, 그에 맞게 타패를 변화시켜 가며 치는 것, 혹은 그런 사상. 반의어: 디지털.
