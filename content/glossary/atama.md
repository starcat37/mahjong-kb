# 머리 (頭 / atama)

| 항목 | 내용 |
|------|------|
| 일본어 | 頭 (アタマ) |
| 로마자 | atama |
| 한국어 | 머리 |

## 정의

머리(頭/아타마)는 화료 형태에서 멘쯔 4개 외에 필요한 같은 패 2장의 쌍(또이쯔)이다.

## 상세 설명

일반적인 화료 형태 '4멘쯔 1또이쯔'에서 또이쯔 부분을 머리라 한다. 핑후(pinfu)에서는 머리가 역패가 아니어야 하며, 머리에 따라 부수가 달라질 수 있다. 삼원패·자풍패·장풍패가 머리인 경우 2부가 추가된다.

## 사용 예시

- "머리를 역패가 아닌 패로 잡아야 핑후가 된다."
- "삼원패 머리는 부수 2부가 추가된다."
