# 간짱대기 (嵌張待ち / kanchan)

| 항목 | 내용 |
|------|------|
| 일본어 | 嵌張待ち (カンチャンまち) |
| 로마자 | kanchan |
| 한국어 | 간짱대기 |

## 정의
간짱대기(嵌張待ち)는 수패의 사이 한 장을 기다리는 대기 형태.

## 상세 설명
예를 들어 3-5를 들고 4를 기다리는 형태. 대기패 1종 최대 4장으로 양면보다 효율이 떨어진다. 부수 계산에서 대기 부수 2부.

## 사용 예시
- "2통-4통으로 3통 간짱대기"
- "간짱대기는 양면대기보다 화료율이 낮다"
