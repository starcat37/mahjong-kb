# 테다시 (手出し / tedashi)

| 항목 | 내용 |
|------|------|
| 일본어 | 手出し |
| 로마자 | tedashi |
| 한국어 | 테다시 |

## 정의

쯔모해 온 패를 손 안에 넣고, 손 안에 있던 패를 버리는 행위.
