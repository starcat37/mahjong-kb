# 평착 (平着 / heichaku)

| 항목 | 내용 |
|------|------|
| 일본어 | 平着 |
| 로마자 | heichaku |
| 한국어 | 평착 |

## 정의

'평균 착순'의 약어. (1위 횟수×1 + 2위 횟수×2 + 3위 횟수×3 + 4위 횟수×4) ÷ 대전수라는 공식을 통해 구할 수 있다.
