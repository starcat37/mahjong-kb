# 쵼보 (チョンボ / chonbo)

| 항목 | 내용 |
|------|------|
| 일본어 | チョンボ |
| 로마자 | chonbo |
| 한국어 | 쵼보 |

## 정의

쵼보(チョンボ)는 마작에서의 반칙으로, 역 없는 화료 선언이나 규칙 위반 시 부과되는 벌칙.

## 상세 설명

대표적인 쵼보는 역 없이 론/쯔모를 선언하는 것이다. 쵼보 발생 시 만관 벌금을 지불한다(자: 8000점, 친: 12000점). 해당 국은 무효화되어 같은 국을 다시 진행하며, 본장은 늘어나지 않는다.

## 사용 예시

- "역이 없는데 론을 선언하면 쵼보 만관 벌금"
- "쵼보가 발생하면 그 국은 무효가 된다"

## 관련 용어

- [[agari]] — 잘못된 화료 선언이 쵼보의 주요 원인
- [[ron]] — 역 없이 론 선언 시 쵼보
- [[mangan]] — 쵼보 벌금은 만관 상당

- [[fu]] — 쵼보와 무관하게 만관 고정 벌금
