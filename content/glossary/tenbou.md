# 점봉 (点棒 / tenbou)

| 항목 | 내용 |
|------|------|
| 일본어 | 点棒 |
| 로마자 | tenbou |
| 한국어 | 점봉 |

## 정의

화료에 의해 점수의 이동이 있을 때 주고받는 봉. 각각의 봉은 점의 개수로 구별한다.
