# 오리 (降り / ori)

| 항목 | 내용 |
|------|------|
| 일본어 | 降り |
| 로마자 | ori |
| 한국어 | 오리 |

## 정의

자신의 손을 갖추는 것보다 타가에게 방총하지 않는 것을 우선하여, 안전한 패를 버리거나 위험패의 타패를 피하는 전법. 화료를 완전히 포기하지는 않는다는 점에서 베타오리와 구별된다.
