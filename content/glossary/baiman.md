# 배만 (倍満 / baiman)

| 항목 | 내용 |
|------|------|
| 일본어 | 倍満 (ばいまん) |
| 로마자 | baiman |
| 한국어 | 배만 |

## 정의

배만(倍満)은 8~10판일 때 적용되는 점수 단계로, 만관의 2배.

## 상세 설명

자(子) 론 16000점, 쯔모 4000-8000점. 친(親) 론 24000점, 쯔모 8000올. 혼일색(6판) + 여러 역의 복합, 또는 청일색(6판) + 도라 등으로 도달 가능하다. 배만 이상은 실전에서 드물게 나온다.

## 사용 예시

- "청일색 6판 + 도라3이면 9판 배만"
- "배만 이상은 실전에서 드물게 나온다"
