# 아시도메 (足止め / ashidome)

| 항목 | 내용 |
|------|------|
| 일본어 | 足止め |
| 로마자 | ashidome |
| 한국어 | 아시도메 |

## 정의

타가의 손 진행을 방해할 목적으로 건 리치. 대기가 나쁘거나 타점이 싸구려라도 방해를 위해 거는 경우를 지칭.
