# 벌칙

## 개요

리치마작 대국 중 규칙을 위반했을 때 적용되는 벌칙 규정을 정리한다. 벌칙은 크게 **화료 불가**(아가리 불가)와 **쵼보(チョンボ)**의 두 단계로 나뉘며, 위반의 심각도에 따라 다르게 적용된다.

## 상세 규칙

### 화료 불가(アガリ放棄)

화료 불가 처분을 받은 플레이어에게 적용되는 제한사항:

1. 해당 국에서 **화료할 수 없다**.
2. **모든 발성**(치, 퐁, 깡, 리치, 쯔모, 론)을 할 수 없으며, 이를 어길 시 쵼보 취급한다.
3. 유국 시 **노텐(ノーテン) 취급**한다.

#### 화료 불가가 적용되는 경우

| 상황 | 설명 |
|------|------|
| 치/퐁/깡 발성 후 손패 공개했으나 실수 | 울기 발성 후 공개한 손패가 잘못된 경우 |
| 쯔모/론 발성 후 손패 미공개 | 화료 선언 후 손패를 보여주지 않은 경우 |
| 쿠이카에(울어바꿈) 적발 | 이미 있는 슌쯔에 치를 부른 후 같은 패나 스지(3칸 차이) 패를 버리는 행위. 룰에 따라 스지 쿠이카에를 허용하거나 현물 쿠이카에를 쵼보로 취급하기도 한다. |
| 다패(多牌) / 소패(少牌) | 손패가 규정보다 많거나 적은 경우 |
| 후리텐 론 (손패 미공개) | 후리텐 상태에서 론을 외쳤으나 손패를 공개하지 않은 경우 |

### 쵼보(チョンボ)

쵼보는 중대한 규칙 위반에 대한 벌칙으로, 위반자가 **만관(満貫)에 해당하는 점수**를 지불해야 한다.

- **반칙자가 친인 경우**: 나머지 자(子) 3명에게 각 **4,000점**씩 지불 (총 12,000점)
- **반칙자가 자인 경우**: 친에게 **4,000점**, 나머지 자 2명에게 각 **2,000점** 지불 (총 8,000점)
- 쵼보 발생 시 해당 국을 **다시 진행**한다 (주사위는 다시 굴리지 않음).
- **연장봉(本場)은 증가하지 않는다**.

> 쵼보의 벌칙 점수는 룰에 따라 차이가 있다. 일부 룰에서는 모든 플레이어에게 각 3,000점(총 9,000점)을 지불하는 방식을 채택한다.

#### 쵼보가 적용되는 경우

| 상황 | 설명 |
|------|------|
| 거짓 화료 | 쯔모/론 발성 후 손패를 공개했으나 실제로 화료가 아닌 경우 |
| 후리텐 론 (손패 공개) | 후리텐 상태에서 론을 외치고 손패를 공개한 경우 |
| 리치 노텐 | 리치 선언 후 유국 시 텐파이가 아닌 것이 확인된 경우 |
| 화료 불가 상태에서 발성 | 이미 화료 불가 처분을 받은 자가 발성한 경우 |
| 리치 후 안깡으로 대기 변동 | 리치 이후 안깡으로 대기패가 변경된 경우 |
| 패산 6장 이상 공개 | 실수로 패산의 패가 6장 이상 공개된 경우 |
| 왕패 2장 이상 공개 | 실수로 왕패가 2장 이상 공개된 경우 |
| 타가 패 2장 이상 공개 | 실수로 다른 플레이어의 패를 2장 이상 공개한 경우 |

### 쵼보 면책

타가의 정당한 화료가 있는 경우, 쵼보 벌칙이 **면책**된다. 즉, 같은 시점에 다른 플레이어가 정당하게 화료했다면 쵼보가 적용되지 않는다.

### 기타 실수 처리

| 상황 | 처리 |
|------|------|
| 패산 패 공개 (6장 미만) | 대국 속행 |
| 왕패 공개 (1장) | 모든 대국자에게 해당 패 공개 후 대국 속행 |
| 비정상 위치에서 쯔모 지적 | 즉시 해당 패를 타패 |

### 1,000점 벌점(小反則)

경미한 반칙에 대해 1,000점의 벌점을 지불하는 경우가 있다. 지불한 1,000점은 다음 화료자가 가져간다.

| 상황 | 설명 |
|------|------|
| 후로 발성 실수 | 치, 퐁, 깡의 발성을 잘못한 경우 |
| 패산 무너뜨리기 | 실수로 패산을 무너뜨린 경우 (공개된 패 1장당 1,000점) |

## 예외 사항

- 쵼보 시에도 타가의 정당한 화료가 있으면 면책된다.
- 치/퐁/깡 발성 후 손패를 아직 공개하지 않은 상태라면, 1000점 공탁 후 발성을 취소할 수 있다.

## 예시

### 거짓 화료(쵼보) 예시
플레이어가 "론"을 외치고 손패를 공개했으나, 실제로 역이 없는 상태였다. 이 경우 쵼보로 처리되어 다른 3명에게 각 3000점을 지불하고, 해당 국을 처음부터 다시 진행한다.

### 화료 불가 예시
플레이어가 "퐁"을 외치고 손패를 공개했으나, 실제로는 퐁할 수 없는 패였다. 화료 불가 처분을 받아, 해당 국이 끝날 때까지 어떤 발성도 할 수 없고 화료도 불가하다. 유국 시 노텐 취급된다.

### 다패/소패 예시
쯔모 순서 실수로 손패가 14장이 되어버린 경우(다패), 즉시 화료 불가 처분이 내려진다. 이후 매 턴 쯔모 없이 타패만 하여 정상 장수로 돌아가야 한다.

## 마작 매너

벌칙 규정 외에도, 원활한 대국을 위해 지켜야 할 매너가 있다.

- **장고(長考)**: 지나치게 긴 고민은 다른 플레이어를 기다리게 하므로, 적절한 시간 내에 판단한다.
- **발성**: 치, 퐁, 깡, 리치, 쯔모, 론 등의 발성은 짧고 또렷하게 한다.
- **패 다루기**: 타인의 패를 소중하게 다루며, 타패 시 세게 내려놓지 않는다.
- **국 후 에티켓**: 국이 끝난 후 자신의 손패나 패산의 패를 공개하거나 언급하지 않는다.
- **상호 존중**: 지고 있는 상대를 놀리지 않고, 기분 나쁜 티를 내지 않으며, 정중한 태도를 유지한다.
