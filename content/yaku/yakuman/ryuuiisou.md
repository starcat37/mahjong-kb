# 료이소 (緑一色)

| 항목 | 내용 |
|------|------|
| 일본어 | 緑一色 |
| 로마자 | ryuuiisou |
| 한국어 | 료이소 |
| 판수 | 역만 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

료이소는 초록색 패만으로 손패 전체를 구성하는 역만이다. 사용 가능한 패는 삭수 2, 3, 4, 6, 8과 발(發)의 총 6종류뿐이다. 패 그림의 색상에 기반한 독특한 역으로, 후로해도 역만이 유지된다.

## 성립 조건

1. 손패 전체가 삭수 2, 3, 4, 6, 8, 발(發) 6종류로만 이루어져야 한다.
2. 위 6종 이외의 패가 하나라도 포함되면 성립하지 않는다.
3. 슌쯔는 234만 가능하다.
4. 발(發) 없이 삭수만으로도 성립 가능하다(룰에 따라 다름).

## 예시

### 기본 예시

```
2s3s4s 2s3s4s 6s6s6s 8s8s8s 發發
```
삭수 234 + 삭수 234 + 삭수 666 + 삭수 888 + 발발 머리

```
2s2s2s 3s3s3s 4s4s4s 發發發 6s6s
```
삭수 222 + 삭수 333 + 삭수 444 + 발발발 + 삭수 66 머리

### 주의할 패턴

- 삭수 1, 5, 7, 9는 초록색이 아니므로 사용할 수 없다.
- 슌쯔는 234 한 종류만 가능하다는 점에 주의한다.
- 발(發)을 포함하지 않는 료이소도 인정하는 룰이 일반적이다.

## 전략적 활용

- 사용 가능한 패가 6종뿐이므로 배패 단계에서 판단해야 한다.
- 삭수 2, 3, 4가 많으면 슌쯔 중심, 6, 8이 많으면 커쯔 중심으로 진행한다.
- 발(發)은 야쿠하이이기도 하므로 퐁하면서 자연스럽게 진행할 수 있다.
- 호닛쯔나 친잇쯔(삭수)를 노리다가 해당 패만 남으면 료이소로 전환 가능하다.
