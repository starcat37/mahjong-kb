# 대면 (対面 / toimen)

| 항목 | 내용 |
|------|------|
| 일본어 | 対面 |
| 로마자 | toimen |
| 한국어 | 대면 |

## 정의

자신 기준 12시 방향에 앉은 사람을 지칭하는 표현. '대면(対面)'은 얼굴을 마주보고 있다는 의미에서 유래.
