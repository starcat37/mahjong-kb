# 유용한 링크

## 연습 도구

### 패효율 연습
- https://euophrys.itch.io/mahjong-efficiency-trainer
- 어떤 패를 버려야 가장 효율적인지 연습하는 트레이너. 한국어 지원. 평점 4.9/5.
- **모드 3종**:
  - 효율 트레이너 — 패 구성 개선 연습 (텐파이까지 최적 버림패 비교)
  - 방어 트레이너(Folding) — 상대 공격에 대한 방어 학습
  - 리플레이 분석 — 실제 게임 기록 분석
- **주요 설정 옵션**:
  | 설정 | 기능 |
  |------|------|
  | 타일 제한 | 특정 무늬 제거로 난이도 조절 |
  | 3인 마작 규칙 | 삼마 규칙 적용 |
  | 레드 파이브 | 붉은 5 타일 포함 |
  | 유령 플레이어 | 다른 플레이어 버림 시뮬레이션 |
  | 최적 옵션 표시 | 최선의 버림패 공개 여부 |
- **입력법**: 숫자 + 무늬 기호 (`m` 만수, `p` 통수, `s` 삭수, `z` 자패). 예: `340m778s236677p55z`

### 텐파이 시 대기패 확인 (다면대기 확인)
- https://tenhou.net/2/
- 텐파이 상태의 손패를 입력하면 대기패(아타리패)를 자동으로 계산해주는 도구.
- 다면대기처럼 복잡한 대기 형태를 확인할 때 유용.
- **패 표기법**: 만수 `m`, 통수 `p`, 삭수 `s`, 자패(동남서북백발중) `1234567z`
- **입력 예시**:
  - 국사무쌍: `19m19p19s1234567z6z`
  - 일반 손패: `123m456p789s11z` → 대기패 표시
  - 다면대기: `2344555m234p234s` → 복수 대기패 표시

### 부판계산법 연습
- https://mahjong-ten.com/index.html
- 부수와 판수 계산을 무한 반복 연습할 수 있는 드릴 형식 사이트.
- **주요 기능**:
  - 무한 드릴 — 손패·화료·깡·부로 정보를 보고 점수를 계산하는 문제가 무한 출제
  - 해답 확인 — 풀이 후 정답과 비교 가능
  - 역(야쿠) 학습 — 역 종류와 판수 이론 정리
  - 부(符) 계산 설명 — 부수 산정 규칙 해설
- 앱 버전도 제공.

### 전뇌마장 (CPU 대전 연습)
- https://kobalab.net/majiang/index.html
- CPU 상대로 마작을 치며 연습할 수 있는 웹게임. 별도 설치 없이 브라우저에서 바로 플레이 가능.

### 청일마작 BAMBOO (다면대기 연습)
- https://www.gamedesign.jp/games/bamboo/
- 삭수로만 플레이하는 2인 마작 웹게임. 청일색 다면대기를 익힐 때 유용.

### 마작 입문 강좌 (추천)
- https://skypine.tistory.com/m/category/마작/마작%20입문%20강좌
- 총 18개 강좌로 구성된 체계적 리치마작 입문서.
- **기초 강좌 (1-5강)**: 마작패와 부속품 → 마작의 목표와 멘쯔 만들기 → 조패·쯔모·기리 → 후로(울기) → 대국의 구성
- **중급 강좌 (6-10강)**: 대국의 시작과 패산 쌓기 → 역과 점수 → 리치 및 멘젠 한정 역 → 멘젠 준한정/비한정 역과 역만 → 반칙과 매너
- **특별편 (3강)**: 후리텐과 골라쏘기, 깡, 로컬 룰
- **부록 (5강)**: 부수 계산과 점수표, 1판역 정리, 2판역 정리, 3판 이상 역 정리, 역만 정리

### 마작 초급 강좌 (추천)

- **강좌 목록**:

  3. 패의 가치
  4. 리치 수비하기

  6. 타가의 패 읽기
  7. 리치의 판단과 다마텐
  8. 후로의 판단과 역 없음
  9. 대국적인 대국

## 참고 자료

### 역 목록
- https://namu.wiki/w/리치마작/역
- 나무위키의 리치마작 역 목록.

### 마작 팁 모음 (arca.live)
- https://arca.live/b/majak/6728976
- 아카라이브 마작 채널의 뉴비~중수 대상 글모음. 실력 수준별로 정리되어 있다.
- **초심자용**: 룰/패/족보 설명, 부수 계산법, 용어 정리, 역없음 회피 지침서
- **중급자용 시리즈**: 공수판단(6편), 패효율(8편), 초보자 수비 강의(2편), 히라사와 겐키 번역, 금탁 가는 마작 특강, 14대 천봉위 시리즈
- **중급자용 단편**: 다면대기 형태, 후리텐 리치 활용, 패의 위험도 표, 버림패 읽기, 완전1샨텐, 다마텐 경계법 등
- **문제/해답**: 나니키루(何切る) 시리즈, 리치판단 문제, 무엇을 버릴까요 시리즈 등 풍부한 연습 문제
- **잡지식**: 우마 계산법, 역의 한자 뜻풀이, 통계 기반 역별 출현확률 등

### 점수 암기 팁 (네이버 블로그)
- https://m.blog.naver.com/endtgenesis/30113756072
- 부수와 점수를 효율적으로 외우는 요령을 정리한 글.
- **핵심 암기법**: 30부 론을 기준으로 외우고, 25부(치또이쯔)와 40부는 깔끔한 2배 구조를 활용. 20부/60부/80부/100부는 판수를 올리거나 내려서 변환하는 법칙 소개.
- **쯔모 요령**: 친이 지불하는 금액 = 한 판 낮춘 론 점수, 자 지불은 그 약 절반.

### 3인 마작 정보글 모음
- https://m.dcinside.com/board/riichimahjong/1024
- 리치마작 갤러리의 3인 마작(삼마) 종합 자료 모음.
- **입문 자료**: 삼마의 권우 번역, 우니마루의 삼마초심자강좌, 3인 마작 입문을 위한 이야기, 삼마학개론(2부작)
- **중급·고급 자료**: 삼마공략 번역 모음, 우니마루의 삼마 울기판단 18문제, 삼마공략 3부작
- **추천 저술서 3권**: 《삼인마작의 극의》, 《데이터로 이기는 삼인마작》, 《오니우치 천봉위의 삼인마작 승리의 프로세스》
- **기타**: Discord 커뮤니티, 3인마작 AI 도구 (혼천~천봉위 수준), 기초 테크닉·방총률·공수판단 표 등 실전 자료
