# 장풍 (場風 / bakaze)

| 항목 | 내용 |
|------|------|
| 일본어 | 場風 |
| 로마자 | bakaze |
| 한국어 | 장풍 |

## 정의

그 국에서 장에 주어진 바람을 뜻하는 말. 모든 사람이 공유한다. 동장이면 東, 남장이면 南이 장풍패가 된다.
