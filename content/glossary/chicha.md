# 기가 (起家 / chicha)

| 항목 | 내용 |
|------|------|
| 일본어 | 起家 |
| 로마자 | chicha |
| 한국어 | 기가 |

## 정의

대국을 개시했을 때 최초로 오야 차례인 사람(동 1국의 동가)을 뜻하는 말. 일본어 발음은 '치챠(チーチャ)'.
