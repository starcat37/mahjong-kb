# 타패 (打牌 / dahai)

| 항목 | 내용 |
|------|------|
| 일본어 | 打牌 (だはい) |
| 로마자 | dahai |
| 한국어 | 타패 |

## 정의

타패(打牌)는 쯔모 후 손패에서 불필요한 패 1장을 버리는 행동.

## 상세 설명

쯔모한 패를 그대로 버리는 것을 쯔모기리(ツモ切り)라 한다. 버린 패는 자신의 앞에 순서대로 나열하며(하천/河), 이 정보로 다른 플레이어의 패와 전략을 추측할 수 있다.

## 사용 예시

- 불필요한 9삭을 타패한다
- 쯔모기리는 불요패라는 신호를 줄 수 있다
