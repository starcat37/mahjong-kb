# 샨텐모도시 (向聴戻し / shanten modoshi)

| 항목 | 내용 |
|------|------|
| 일본어 | 向聴戻し |
| 로마자 | shanten modoshi |
| 한국어 | 샨텐모도시 |

## 정의

샨텐 수를 굳이 되돌리는(늘리는) 행위. 주로 이샨텐/량샨텐인 상태에서 손역을 노리거나 최종적으로 양형 대기를 잡기 위해 행해진다.
