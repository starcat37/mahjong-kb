# 라스오야 (ラス親 / rasu oya)

| 항목 | 내용 |
|------|------|
| 일본어 | ラス親 |
| 로마자 | rasu oya |
| 한국어 | 라스오야 |

## 정의

오라스에서 오야 차례인 사람. 꼴찌라도 라스오야를 잡고 묵묵히 연장을 반복해서 결국 역전하는 경우가 있어 남 3국까지의 1위에게는 공포의 대상.
