# 텐파이하즈시 (聴牌外し / tenpai hazushi)

| 항목 | 내용 |
|------|------|
| 일본어 | 聴牌外し |
| 로마자 | tenpai hazushi |
| 한국어 | 텐파이하즈시 |

## 정의

텐파이가 된 상태에서 굳이 텐파이를 잡지 않고 이샨텐으로 되돌아가는 행위.
