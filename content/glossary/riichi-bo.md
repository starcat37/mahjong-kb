# 리치봉 (立直棒 / riichi-bo)

| 항목 | 내용 |
|------|------|
| 일본어 | 立直棒 (リーチぼう) |
| 로마자 | riichi-bo |
| 한국어 | 리치봉 |

## 정의

리치봉(立直棒)은 리치 선언 시 탁상에 내놓는 1000점짜리 점봉.

## 상세 설명

리치를 선언할 때 반드시 1000점을 공탁해야 한다. 해당 국에서 화료한 플레이어가 모든 리치봉을 가져간다. 유국 시 리치봉은 탁상에 남아 다음 국 화료자에게 넘어간다.

## 사용 예시

- 리치 선언과 함께 1000점봉을 탁상에 낸다
- 리치봉이 2개 쌓인 국에서 화료하면 2000점 보너스
