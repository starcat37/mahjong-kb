# 연장 (連荘 / renchan)

| 항목 | 내용 |
|------|------|
| 일본어 | 連荘 (れんちゃん) |
| 로마자 | renchan |
| 한국어 | 연장 |

## 정의

연장(連荘)은 친이 계속 유지되어 같은 국이 반복되는 것.

## 상세 설명

친이 화료하거나, 유국 시 친이 텐파이이면 연장이 된다. 연장 시 본장(本場)이 1씩 올라가며, 화료 점수에 본장당 300점(쯔모는 100점씩 x 3명)이 추가된다. 연장이 계속되면 고득점이 오갈 수 있어 게임의 긴장감이 높아진다.

## 사용 예시

- 친이 3연속 화료하여 3본장까지 갔다
- 연장이 이어질수록 본장 보너스가 쌓인다
