# 쿠이사가리 (食い下がり / kuisagari)

| 항목 | 내용 |
|------|------|
| 일본어 | 食い下がり |
| 로마자 | kuisagari |
| 한국어 | 쿠이사가리 |

## 정의

비멘젠일 경우 역의 판수가 떨어지는 것. e.g.) 일기통관, 삼색동순.
