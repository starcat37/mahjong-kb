# 오카 (オカ / oka)

| 항목 | 내용 |
|------|------|
| 일본어 | オカ |
| 로마자 | oka |
| 한국어 | 오카 |

## 정의

기본 점수와 반환 점수에 차등을 두어, 그 차등점만큼의 점수를 1등에게 몰아주는 룰. e.g.) 25000점 시작 30000점 반환일 경우 차등점 5000점 × 4명 = 20000점을 1위에게 지급.
