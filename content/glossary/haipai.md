# 배패 (配牌 / haipai)

| 항목 | 내용 |
|------|------|
| 일본어 | 配牌 (はいぱい) |
| 로마자 | haipai |
| 한국어 | 배패 |

## 정의

배패(配牌)는 국이 시작될 때 각 플레이어에게 나누어주는 처음 패.

## 상세 설명

친(親)은 14장, 자(子)는 13장을 받는다. 배패는 패산(山)에서 시계 반대 방향으로 4장씩 가져가며 배분. 배패 운이 좋으면 텐파이에 빨리 도달할 수 있다.

## 사용 예시

- 배패에 도라가 3장이나 들어왔다
- 배패 13장을 보고 어떤 역을 노릴지 판단한다
