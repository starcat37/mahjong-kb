# 밍깡 (明槓 / minkan)

| 항목 | 내용 |
|------|------|
| 일본어 | 明槓 (ミンカン) |
| 로마자 | minkan |
| 한국어 | 밍깡 |

## 정의
밍깡(明槓/대명깡)은 손패의 같은 패 3장(안커)에 다른 플레이어의 타패를 더해 선언하는 깡.

## 상세 설명
다른 플레이어가 버린 패를 이용하므로 멘젠이 깨진다. 부수 계산에서 밍깡은 안깡의 절반(중장패 8부, 요구패 16부). 밍깡에 대해 창깡(chankan)이 성립할 수 있다.

## 사용 예시
- "하가가 버린 중을 밍깡"
- "밍깡은 안깡보다 부수가 낮다"
