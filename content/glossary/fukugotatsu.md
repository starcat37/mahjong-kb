# 복합타쯔 (複合塔子 / fukugou tatsu)

| 항목 | 내용 |
|------|------|
| 일본어 | 複合塔子 |
| 로마자 | fukugou tatsu |
| 한국어 | 복합타쯔 |

## 정의

타쯔나 또이쯔가 서로 겹쳐진 채 하나의 블럭을 형성하고 있는 것을 지칭하는 말. e.g.) 112통, 344만, 357삭 등.
