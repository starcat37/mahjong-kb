# 우라스지 (裏筋 / ura suji)

| 항목 | 내용 |
|------|------|
| 일본어 | 裏筋 |
| 로마자 | ura suji |
| 한국어 | 우라스지 |

## 정의

간짱에서 양면으로의 대기 변화를 가정했을 때의 위험패. e.g.) 13에서 4가 붙어 1을 버리는 경우의 2,5.
