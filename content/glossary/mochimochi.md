# 모찌모찌 (持ち持ち / mochi mochi)

| 항목 | 내용 |
|------|------|
| 일본어 | 持ち持ち |
| 로마자 | mochi mochi |
| 한국어 | 모찌모찌 |

1종류의 패를 2명이 2장씩 가지고 있는 경우를 뜻하는 말. 주로 역패인 자패가 대상이 된다.
