# 멘젠쯔모 (門前清自摸和)

| 항목 | 내용 |
|------|------|
| 일본어 | 門前清自摸和 |
| 로마자 | menzen tsumo |
| 한국어 | 멘젠쯔모 |
| 판수 | 1판 |
| 후로 가능 | 불가 |
| 쿠이사가리 | 없음 |

## 개요

멘젠쯔모는 울지 않은 멘젠 상태에서 자신이 직접 쯔모하여 화료하는 역이다. 론이 아닌 쯔모로 완성해야 하며, 가장 기본적인 역 중 하나로 다른 역과 자유롭게 복합된다.

## 성립 조건

1. 멘젠 상태여야 한다 (치, 폰, 대명깡을 하지 않은 상태).
2. 자신의 쯔모로 화료해야 한다 (론 화료는 해당하지 않음).

## 예시

### 기본 예시

만수 123, 통수 456, 삭수 789, 삭수 11, 통수 55 — 통수 5를 쯔모하여 커쯔를 완성하면 멘젠쯔모 성립.

### 주의할 패턴

- 한 번이라도 치, 폰, 대명깡을 했다면 멘젠이 깨져 이 역은 성립하지 않는다.
- 암깡은 멘젠 상태를 유지하므로 멘젠쯔모와 복합 가능하다.

## 전략적 활용

- 리치와 함께 선언하면 리치 + 멘젠쯔모로 최소 2판이 확보된다.
- 멘젠을 유지하는 것 자체가 판수를 올리는 데 유리하므로, 가능하면 울지 않고 손패를 완성하는 연습이 중요하다.
- 쯔모 화료는 3명에게 점수를 분산 지불받으므로, 론보다 실질 수익이 높을 수 있다.
