# 대삼원 (大三元)

| 항목 | 내용 |
|------|------|
| 일본어 | 大三元 |
| 로마자 | daisangen |

## 개요

대삼원은 삼원패(백, 발, 중) 3종을 모두 몸통(커쯔 또는 깡쯔)으로 갖추는 역만이다. 후로해도 역만이 유지되며, 책임지불(파오) 규칙이 적용되는 대표적인 역이다.

## 성립 조건

1. 백(白), 발(發), 중(中) 3종 모두를 커쯔 또는 깡쯔로 갖춰야 한다.
2. 나머지 1몸통과 머리 1조를 자유롭게 구성한다.
3. 멘젠, 후로 모두 가능하며 판수는 동일하다.

## 책임지불 (파오)

- 이미 삼원패 중 2종을 퐁(또는 깡)으로 확보한 플레이어에게 3번째 삼원패를 버려 퐁시킨 자가 **책임지불** 대상이 된다.
- 쯔모 화료 시: 책임지불자가 전액 부담.
- 론 화료 시: 방총자와 책임지불자가 절반씩 부담.

## 예시

### 기본 예시

```
白白白 發發發 中中中 3m4m5m 7p7p
```
백백백 + 발발발 + 중중중 + 만수 345 + 통수 77 머리

```
白白白 發發發 中中中 1s1s1s 9m9m
```
백백백 + 발발발 + 중중중 + 삭수 111 + 만수 99 머리

### 주의할 패턴

- 삼원패 중 2종만 몸통이고 1종이 머리이면 쇼산겐(소삼원, 2판)이 된다.
- 3번째 삼원패를 퐁시키는 행위는 책임지불 대상이 되므로, 상대의 삼원패 2종 노출 시 3번째를 버리지 않도록 주의한다.

## 전략적 활용

- 배패에 삼원패가 다수 있을 때 적극적으로 노린다.
- 2종의 삼원패를 퐁한 플레이어에게는 나머지 삼원패를 절대 버리지 않는 것이 수비의 기본이다.
- 후로해도 역만이므로 속공형으로 운영할 수 있다.
