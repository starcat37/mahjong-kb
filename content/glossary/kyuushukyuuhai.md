# 구종구패 (九種九牌 / kyuushu kyuuhai)

| 항목 | 내용 |
|------|------|
| 일본어 | 九種九牌 |
| 로마자 | kyuushu kyuuhai |
| 한국어 | 구종구패 |

## 정의

배패를 받고 첫 순 자신의 패에 포함된 요구패가 9종 이상일 경우 손패를 공개하고 도중유국을 시킬 수 있는 로컬 룰. 자신의 첫 쯔모 전에 타가의 발성(치, 펑, 깡)이 있을 경우 선언할 수 없다. 대부분의 넷마작에서 채택하지만, 프로경기에서는 도중유국 자체를 인정하지 않는 경우가 많다.
