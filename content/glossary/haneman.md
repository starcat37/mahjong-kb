# 하네만 (跳満 / haneman)

| 항목 | 내용 |
|------|------|
| 일본어 | 跳満 (はねまん) |
| 로마자 | haneman |
| 한국어 | 하네만 |

## 정의

하네만(跳満)은 6~7판일 때 적용되는 점수 단계로, 만관의 1.5배.

## 상세 설명

자(子) 론 12000점, 쯔모 3000-6000점. 친(親) 론 18000점, 쯔모 6000올. 리치 쯔모에 뒷도라나 도라가 여러 개 실리면 하네만에 도달하는 경우가 많다.

## 사용 예시

- "7판이면 하네만 12000점(자 론 기준)"
- "리치 쯔모 이페코 뒷도라2로 하네만!"
