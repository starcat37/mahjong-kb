# 스지 (筋 / suji)

| 항목 | 내용 |
|------|------|
| 일본어 | 筋 (スジ) |
| 로마자 | suji |
| 한국어 | 스지 |

## 정의

스지(筋)는 양면 대기의 원리를 이용한 방어 기법으로, 특정 패가 버려졌을 때 그 패와 양면 관계에 있는 패가 비교적 안전하다고 판단하는 논리이다.

## 상세 설명

수패의 스지 관계는 1-4-7, 2-5-8, 3-6-9의 세 쌍으로 구성된다. 예를 들어 4가 버려져 있다면, 양면 대기(2-3 대기 또는 5-6 대기)에서 4로 론할 수 없으므로 1과 7이 비교적 안전하다고 판단한다. 다만 스지는 양면 대기에 대해서만 유효하며, 샤보(shanpon)나 간짱(kanchan) 대기에는 통하지 않으므로 과신은 금물이다.

## 사용 예시

- "4통이 버려져 있으니 1통과 7통은 스지라서 비교적 안전하다."
- "스지만 믿고 버리면 간짱이나 샤보에 걸릴 수 있다."
