# 멘젠 (門前 / menzen)

| 항목 | 내용 |
|------|------|
| 일본어 | 門前 (メンゼン) |
| 로마자 | menzen |
| 한국어 | 멘젠 |

## 정의

멘젠(門前)은 치·퐁·밍깡 등의 울기를 하지 않고 자력으로만 패를 모은 상태이다.

## 상세 설명

멘젠 상태에서만 리치(riichi) 선언이 가능하며, 쯔모 화료 시 멘젠쯔모(tsumo) 역이 성립한다. 멘젠은 패효율은 떨어지지만 역의 선택지가 넓고 뒷도라 기회가 있어 일반적으로 높은 점수를 기대할 수 있다.

## 사용 예시

- "멘젠을 유지해야 리치를 걸 수 있다."
- "멘젠 상태의 쯔모는 1판 역이다."
