# 만관 (満貫 / mangan)

| 항목 | 내용 |
|------|------|
| 일본어 | 満貫 (まんがん) |
| 로마자 | mangan |
| 한국어 | 만관 |

## 정의

만관(満貫)은 5판 또는 부수와 판수의 곱이 일정 이상일 때 적용되는 고정 점수 단계.

## 상세 설명

자(子) 론 8000점, 쯔모 2000-4000점. 친(親) 론 12000점, 쯔모 4000올. 3판 70부 이상 또는 4판 40부 이상도 만관으로 올림하는 룰(절상만관)이 일반적이다. 만관부터는 부수 계산이 필요 없다.

## 사용 예시

- "리치 쯔모 도라3이면 5판 만관!"
- "친 만관 쯔모는 4000점씩 3명에게서"
