# 깡쯔 (槓子 / kantsu)

| 항목 | 내용 |
|------|------|
| 일본어 | 槓子 (カンツ) |
| 로마자 | kantsu |
| 한국어 | 깡쯔 |

## 정의

깡쯔(槓子)는 동일한 패 4장으로 이루어진 특수한 멘쯔이다.

## 상세 설명

깡 선언을 통해 만들며, 멘쯔 1개로 취급한다. 안깡(暗槓)·밍깡(明槓)·가깡(加槓) 세 종류가 있다. 깡을 하면 영상패에서 1장을 추가로 가져오고, 깡도라 표시패가 추가된다.

## 사용 예시

- "같은 패 4장이 모이면 깡을 선언할 수 있다."
- "깡쯔는 멘쯔 하나로 취급하지만 부수가 높다."
