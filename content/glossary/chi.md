# 치 (チー / chi)

| 항목 | 내용 |
|------|------|
| 일본어 | チー |
| 로마자 | chi |
| 한국어 | 치 |

## 정의
치(チー)는 왼쪽(상가) 플레이어가 버린 패를 가져와 슌쯔를 만드는 울기.

## 상세 설명
오직 상가(왼쪽)의 타패만 치할 수 있다. 퐁이나 깡보다 우선순위가 낮아 다른 플레이어가 퐁/깡/론을 선언하면 치는 취소된다. 치를 하면 멘젠이 깨진다.

## 사용 예시
- "상가가 버린 5통을 치하여 4-5-6통 슌쯔 완성"
- "치는 상가에서만 가능하다"
