# 다이밍깡 (大明槓 / daiminkan)

| 항목 | 내용 |
|------|------|
| 일본어 | 大明槓 |
| 로마자 | daiminkan |
| 한국어 | 다이밍깡 |

## 정의

안커를 모은 상태에서 상대가 버린 패를 발성으로 가져와 만든 깡쯔.
