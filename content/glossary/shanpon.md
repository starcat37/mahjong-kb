# 샤보대기 (シャボ待ち / shanpon)

| 항목 | 내용 |
|------|------|
| 일본어 | シャボ待ち (シャボまち) |
| 로마자 | shanpon |
| 한국어 | 샤보대기 |

## 정의
샤보대기(シャボ待ち)는 또이쯔 2개를 들고 어느 한쪽이 커쯔가 되기를 기다리는 대기 형태.

## 상세 설명
3멘쯔 + 또이쯔 2개 상태에서 어느 쪽이든 1장이 오면 커쯔+머리로 화료. 대기패 2종 최대 4장(각 2장). 양면과 대기패 수는 비슷하지만 실질적 유효패는 더 적다.

## 사용 예시
- "백 2장과 1만 2장의 샤보대기"
- "샤보대기는 두 종류의 패 모두 론·쯔모 가능"
