# 오쿠리 깡 (送り槓 / okuri kan)

| 항목 | 내용 |
|------|------|
| 일본어 | 送り槓 |
| 로마자 | okuri kan |
| 한국어 | 오쿠리 깡 |

## 정의

깡의 대상이 되는 패 이외의 패를 쯔모한 후, 안깡을 치는 행위. 리치 후의 오쿠리 깡은 주류 룰에서는 반칙 행위에 해당된다.
