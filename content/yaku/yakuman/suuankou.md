# 스안커 (四暗刻)

| 항목 | 내용 |
|------|------|
| 일본어 | 四暗刻 |
| 로마자 | suuankou |
| 한국어 | 스안커 |
| 판수 | 역만 |
| 후로 가능 | 불가 (멘젠만) |
| 쿠이사가리 | 해당 없음 |

## 개요

스안커는 안커(멘젠 상태에서 만든 커쯔) 4개와 머리 1조로 구성하는 역만이다. 또이또이의 멘젠 완전 상위 호환으로, 모든 몸통을 울지 않고 직접 모아야 한다. 단기대기(머리 대기)로 쯔모 화료하면 더블역만으로 인정하는 룰이 있다.

## 성립 조건

1. 멘젠(문전) 상태여야 한다.
2. 안커(암각) 4개를 갖춰야 한다.
3. 머리(작두) 1조가 필요하다.
4. 쯔모 화료가 기본이다.

## 예시

### 기본 예시

```
1m1m1m 5p5p5p 9s9s9s 東東東 白白
```
만수 111 + 통수 555 + 삭수 999 + 동동동 (모두 안커) + 백백 머리

```
3m3m3m 7m7m7m 2p2p2p 中中中 6s6s
```
만수 333 + 만수 777 + 통수 222 + 중중중 (모두 안커) + 삭수 66 머리

### 주의할 패턴

- **샤보대기(양면 커쯔 대기)에서 론으로 화료하면 스안커가 아닌 산안코 + 또이또이가 된다.** 론으로 완성된 몸통은 밍커(명각)로 취급되기 때문이다.
- 샤보대기에서 쯔모로 화료하면 스안커가 성립한다.
- 단기대기(머리 대기)에서는 론으로 화료해도 스안커가 성립한다.
- 단기대기 쯔모 시 더블역만(스안커 단기)으로 인정하는 룰이 있다.

## 전략적 활용

- 배패에 또이쯔(대자)가 3~4조 있을 때 노려볼 수 있다.
- 안커 3개가 완성되면 남은 1조의 대기 형태가 중요하다. 단기대기를 만들면 론으로도 역만이 확정된다.
- 샤보대기가 되면 쯔모에 의존해야 하므로 위험도가 높아진다.
- 안깡은 안커로 인정되므로, 안깡을 포함해도 스안커가 성립한다.
