# 량항시바리 (両翻縛り / ryouhan shibari)

| 항목 | 내용 |
|------|------|
| 일본어 | 両翻縛り |
| 로마자 | ryouhan shibari |
| 한국어 | 량항시바리 |

## 정의

화료에 필요한 판수를 최소 2판으로 규정하는 룰. 기본적으로는 잘 채용되지 않지만, 5연장 이상부터 량항시바리가 적용되는 룰은 대중적으로 쓰인다.
