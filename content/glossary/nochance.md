# 노찬스 (ノーチャンス / no chance)

| 항목 | 내용 |
|------|------|
| 일본어 | ノーチャンス |
| 로마자 | no chance |
| 한국어 | 노찬스 |

## 정의

한 종류의 수패가 벽(4장이 전부 보이는 상태)이 된 경우, 해당 수패를 포함한 대기는 만들 수 없다는 논리적 추론을 이용한 수비 이론. 주로 123, 789와 같은 끝쪽 수패에 적용한다. 우형대기(단기, 샤보 등)에는 쏘일 수 있다.
