# 겐바리 (現張り / genbari)

| 항목 | 내용 |
|------|------|
| 일본어 | 現張り |
| 로마자 | genbari |
| 한국어 | 겐바리 |

## 정의

리치를 건 사람의 현물, 혹은 눈에 띄는 후로를 해 놓은 사람의 현물로 대기를 잡아 텐파이하고 있는 것.
