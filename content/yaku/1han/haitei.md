# 하이테이 (海底摸月)

| 항목 | 내용 |
|------|------|
| 일본어 | 海底摸月 |
| 로마자 | haitei raoyue |
| 한국어 | 해저로월 |
| 판수 | 1판 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

하이테이는 벽패의 마지막 쯔모패로 화료하는 역이다. 한 국에서 단 한 번만 발생할 수 있는 우연역으로, 의도적으로 노리기는 어렵지만 텐파이를 유지하는 것만으로 기회가 생긴다.

## 성립 조건

1. 벽패의 마지막 쯔모패(해저패)로 쯔모 화료해야 한다.
2. 다른 역의 유무와 관계없이, 해저패로 쯔모하면 자동으로 1판이 추가된다.

## 예시

### 기본 예시

벽패가 마지막 1장 남은 상황에서 자신의 쯔모 차례가 돌아왔고, 그 패가 대기패와 일치하여 쯔모 화료 — 하이테이 성립.

### 주의할 패턴

- 마지막 쯔모패를 가져온 사람만 하이테이 대상이다. 그 패를 버렸을 때 다른 사람이 론하면 하이테이가 아닌 호테이가 된다.
- 영상패(깡 후 가져오는 패)는 해저패로 취급하지 않는다.

## 전략적 활용

- 의도적으로 노리기 어려운 역이지만, 국 종반까지 텐파이를 유지하면 자연스럽게 기회가 생긴다.
- 종반에 텐파이를 유지하는 것 자체가 노텐 벌점 회피와 하이테이 가능성 확보라는 이중 이점이 있다.
- 초보자는 "마지막 패로 쯔모하면 1판 추가"라고만 기억해도 충분하다.
