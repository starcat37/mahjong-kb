# 평장 (平場 / heijou)

| 항목 | 내용 |
|------|------|
| 일본어 | 平場 |
| 로마자 | heijou |
| 한국어 | 평장 |

## 정의

① 연장봉이 쌓여있지 않은 상황, 즉 0본장. ② 크게 앞서나가거나 뒤쳐진 사람 없이 점수 상황이 전부 비슷한 상태.
