# 삭수패 (索子 / souzu)

| 항목 | 내용 |
|------|------|
| 일본어 | 索子 (ソーズ) |
| 로마자 | souzu |
| 한국어 | 삭수패 |

## 정의

삭수패는 리치마작에서 사용하는 세 가지 수패 중 하나로, 대나무 무늬가 새겨져 있으며 1삭부터 9삭까지 각 4장씩 총 36장으로 구성된다.

## 상세 설명

1삭은 새(공작) 그림으로 표현되며, 나머지는 대나무 막대기의 개수로 숫자를 나타낸다. 일본어로 소즈(ソーズ)라고 불린다. 녹일색(ryuuiisou)은 삭수패 2, 3, 4, 6, 8과 발(發)로만 구성되는 역만이다.

## 사용 예시

- "삭수패 연속 3장으로 슌쯔를 완성했다."
- "녹일색은 삭수패 중심의 역만이다."
