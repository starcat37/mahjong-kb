# 쇼스시 (小四喜)

| 항목 | 내용 |
|------|------|
| 일본어 | 小四喜 |
| 로마자 | shousuushii |
| 한국어 | 쇼스시 |
| 판수 | 역만 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

쇼스시는 풍패(동, 남, 서, 북) 4종 중 3종을 몸통(커쯔 또는 깡쯔)으로, 나머지 1종을 머리로 구성하는 역만이다. 다이스시(대사희)의 하위 호환으로, 풍패 하나가 머리라는 점이 차이이다.

## 성립 조건

1. 풍패 4종(東, 南, 西, 北) 중 3종을 커쯔 또는 깡쯔로 갖춰야 한다.
2. 나머지 1종의 풍패를 머리(작두)로 사용한다.
3. 남은 1몸통은 수패 또는 자패로 자유롭게 구성한다.
4. 멘젠, 후로 모두 가능하며 판수는 동일하다.

## 예시

### 기본 예시

```
東東東 南南南 西西西 3p4p5p 北北
```
동동동 + 남남남 + 서서서 + 통수 345 + 북북 머리

```
東東東 南南南 北北北 7s8s9s 西西
```
동동동 + 남남남 + 북북북 + 삭수 789 + 서서 머리

### 주의할 패턴

- 풍패 4종 모두를 몸통으로 만들면 다이스시(대사희)가 되며, 더블역만으로 인정하는 룰이 있다.
- 풍패 3종이 몸통이고 머리가 풍패가 아니면 쇼스시가 성립하지 않는다.
- 쯔이소(자일색)와 복합 가능하다.

## 전략적 활용

- 배패에 풍패가 다수 집중되어 있을 때 노릴 수 있다.
- 풍패 2종을 퐁한 상태에서 나머지 풍패도 모이면 적극적으로 진행한다.
- 후로해도 역만이므로 속공이 가능하지만, 풍패 대량 노출은 상대에게 경계심을 준다.
- 다이스시로 발전할 가능성도 열어두면서 진행하는 것이 좋다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 쇼스시가 포함되어 있을 확률은 약 **0.010%**이다.
