# 노미 (ノミ / nomi)

| 항목 | 내용 |
|------|------|
| 일본어 | ノミ |
| 로마자 | nomi |
| 한국어 | 노미 |

## 정의

그 역 외에는 성립하고 있지 않다는 의미의 접미어. 리치노미, 핑후노미 등으로 쓴다. 해당 역을 구성하는 손패 안에 도라가 있다면 쓰지 않는다.
