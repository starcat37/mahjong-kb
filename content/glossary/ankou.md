# 안커 (暗刻 / ankou)

| 항목 | 내용 |
|------|------|
| 일본어 | 暗刻 (アンコ) |
| 로마자 | ankou |
| 한국어 | 안커 |

## 정의

안커(暗刻)는 울기 없이 자력으로 모은 커쯔(동일한 패 3장)이다.

## 상세 설명

안커는 밍커에 비해 부수 계산에서 높은 점수를 받는다(중장패 안커 8부, 요구패 안커 16부). 안커 3개면 삼안커(sanankou), 4개면 사안커(suuankou, 역만)가 성립한다. 멘젠 상태에서 쯔모로 완성한 커쯔만 안커로 인정된다.

## 사용 예시

- "안커 3개를 만들어 삼안커를 노린다."
- "론으로 완성한 마지막 커쯔는 안커가 아니라 밍커로 취급한다."
