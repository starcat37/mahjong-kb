# 아이다욘켄 (間四軒 / aida yonken)

| 항목 | 내용 |
|------|------|
| 일본어 | 間四軒 |
| 로마자 | aida yonken |
| 한국어 | 아이다욘켄 |

## 정의

우라스지가 2개 겹친 상태를 뜻하는 말. '같은 색으로 수패가 4칸만큼 떨어져서 버려져 있을 때, 그 안쪽의 스지'라고 기억한다. e.g.) 버림패에 2와 7이 버려져 있을 때의 3-6.
