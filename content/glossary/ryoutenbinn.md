# 료텐빙 (両天秤 / ryoutenbin)

| 항목 | 내용 |
|------|------|
| 일본어 | 両天秤 |
| 로마자 | ryoutenbin |
| 한국어 | 료텐빙 |

## 정의

여러 역의 가능성을 염두에 두고 패를 진행하는 것. '텐빙'이라고만 쓰기도 한다. 직역하면 '양천칭'.
