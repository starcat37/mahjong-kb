# 커쯔 (刻子 / koutsu)

| 항목 | 내용 |
|------|------|
| 일본어 | 刻子 (コーツ) |
| 로마자 | koutsu |
| 한국어 | 커쯔 |

## 정의

커쯔(刻子)는 동일한 패 3장으로 이루어진 멘쯔이다.

## 상세 설명

직접 모은 커쯔는 안커(暗刻), 퐁(ポン)으로 만든 커쯔는 밍커(明刻)라 한다. 안커는 부수 계산에서 밍커보다 높은 점수를 받는다. 또이또이(toitoi)는 커쯔 4개로 구성해야 성립하는 역이다.

## 사용 예시

- "동 3장을 모아 커쯔를 만들었다."
- "퐁을 하면 밍커가 되어 멘젠이 깨진다."
