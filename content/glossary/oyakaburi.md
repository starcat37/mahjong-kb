# 오얏카부리 (親っ被り / oya kaburi)

| 항목 | 내용 |
|------|------|
| 일본어 | 親っ被り |
| 로마자 | oya kaburi |
| 한국어 | 오얏카부리 |

## 정의

오야 외의 사람이 쯔모로 화료할 경우, 오야는 나머지 가들이 부담하는 점수의 2배를 부담하는 것.
