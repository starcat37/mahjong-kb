# 바조로 (場ゾロ / bazoro)

| 항목 | 내용 |
|------|------|
| 일본어 | 場ゾロ |
| 로마자 | bazoro |
| 한국어 | 바조로 |

## 정의

점수 계산시에 붙는 가상의 2판. '판의 수만큼 부에 2를 곱할 때 반드시 2판을 더해서 계산한다'의 그 2판이다. 화료했을 때의 점수를 최소 1000점으로 하기 위해 있는 룰.
