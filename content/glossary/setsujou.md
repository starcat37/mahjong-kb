# 절상 만관 (切り上げ満貫 / kiriage mangan)

| 항목 | 내용 |
|------|------|
| 일본어 | 切り上げ満貫 |
| 로마자 | kiriage mangan |
| 한국어 | 절상 만관 |

## 정의

4판 30부(자 7700, 친 11600)를 만관으로 계산하는 룰. M리그에서 이 룰을 채택하고 있다.

- [[fu]] — 부
