# 모로힛카케 (モロ引っ掛け / moro hikkake)

| 항목 | 내용 |
|------|------|
| 일본어 | モロ引っ掛け |
| 로마자 | moro hikkake |
| 한국어 | 모로힛카케 |

## 정의

리치 선언패와 스지 관계인 패로 대기하는 것. 량칸이 마지막으로 남은 상태로 텐파이하여 한 장을 자르면서 리치할 때 주로 발생한다.
