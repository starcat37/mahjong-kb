# 만수패 (萬子 / manzu)

| 항목 | 내용 |
|------|------|
| 일본어 | 萬子 (マンズ) |
| 로마자 | manzu |
| 한국어 | 만수패 |

## 정의

만수패는 리치마작에서 사용하는 세 가지 수패 중 하나로, 1만(一萬)부터 9만(九萬)까지 각 4장씩 총 36장으로 구성된다.

## 상세 설명

만수패는 한자 '萬(만)'이 새겨진 패로, 숫자를 나타내는 한자와 함께 표기된다. [[pinzu]](통수패), [[souzu]](삭수패)와 함께 수패(数牌)를 구성하며, 슌쯔(순자)나 커쯔(각자)를 만들 수 있다. 3인 마작(산마)에서는 2만~8만이 제거되어 1만과 9만만 사용한다.

## 사용 예시

- "만수패의 4, 5, 6으로 슌쯔를 만들었다."
- "3인 마작에서는 만수패 중장패(2~8)가 빠진다."
