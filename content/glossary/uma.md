# 우마 (ウマ / uma)

| 항목 | 내용 |
|------|------|
| 일본어 | ウマ |
| 로마자 | uma |
| 한국어 | 우마 |

## 정의

대국 종료 후 순위에 따라 점수를 주고받는 행위. 3등이 2등에게, 4등이 1등에게 지불하는 형식이며 5-10, 10-20, 10-30, 20-30 등이 있다.
