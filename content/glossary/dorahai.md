# 도라표시패 (ドラ表示牌 / dora hyoujihai)

| 항목 | 내용 |
|------|------|
| 일본어 | ドラ表示牌 |
| 로마자 | dora hyoujihai |
| 한국어 | 도라표시패 |

## 정의

왕패에서 모두가 볼 수 있도록 뒤집어 놓은 패. 깡을 칠 때마다 한장씩 추가로 뒤집으며 최대 4번까지 뒤집을 수 있다. 도라표시패의 다음 패가 도라가 된다. 수패는 1→2→...→9→1 순서로 순환하며, 자패는 東→南→西→北→東 / 白→發→中→白 순서로 순환한다.
