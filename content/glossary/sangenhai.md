# 삼원패 (三元牌 / sangenhai)

| 항목 | 내용 |
|------|------|
| 일본어 | 三元牌 (サンゲンパイ) |
| 로마자 | sangenhai |
| 한국어 | 삼원패 |

## 정의

삼원패는 백(白/하쿠), 발(發/하츠), 중(中/츙) 3종류의 자패를 말한다.

## 상세 설명

각 삼원패는 4장씩 총 12장이다. 삼원패의 커쯔 하나만 만들어도 역패(yakuhai)가 성립하므로 활용도가 높다. 대삼원(daisangen)은 3종 모두 커쯔, 소삼원(shousangen)은 2종 커쯔+1종 또이쯔로 구성하는 역이다.

## 사용 예시

- "백 커쯔 하나만으로도 역패 1판이 확정된다."
- "삼원패 2종이 보이면 대삼원을 경계해야 한다."
