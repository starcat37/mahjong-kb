# 객풍패 (客風牌 / otakaze)

| 항목 | 내용 |
|------|------|
| 일본어 | 客風牌 |
| 로마자 | otakaze |
| 한국어 | 객풍패 |

## 정의

자신에게 있어 장풍패나 자풍패에 속하지 않는 풍패를 가리키는 말. 3장 모아도 그 자체로는 역이 성립하지 않지만, 또이쯔를 머리로 쓸 때 핑후의 조건을 깨지도 않는다.
