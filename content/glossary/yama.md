# 패산 (山 / yama)

| 항목 | 내용 |
|------|------|
| 일본어 | 山 (やま) |
| 로마자 | yama |
| 한국어 | 패산 |

## 정의

패산(山)은 게임 시작 시 뒤집어 쌓아놓은 패 더미로, 각 플레이어 앞에 2단으로 쌓는다.

## 상세 설명

총 136장(3인 마작은 108장)을 4명이 나누어 각 17묶(2단 34장)씩 쌓는다. 주사위로 시작점을 정한 후 패산에서 순서대로 배패하고 쯔모한다. 패산의 끝 14장(7묶)은 왕패로 분리한다.

## 사용 예시

- 패산에서 시계 반대 방향으로 쯔모한다
- 패산이 왕패까지 다 소진되면 유국
