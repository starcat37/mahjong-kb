# 급소 (急所 / kyuusho)

| 항목 | 내용 |
|------|------|
| 일본어 | 急所 |
| 로마자 | kyuusho |
| 한국어 | 급소 |

## 정의

특정한 간짱이나 변짱 등 텐파이하기 전에 채우고 싶은, 남았을 때 괴로운 부분을 뜻하는 말. e.g.) 도라표시패가 7만일 때의 68만 간짱.
