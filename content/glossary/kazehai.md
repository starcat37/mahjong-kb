# 풍패 (風牌 / kazehai)

| 항목 | 내용 |
|------|------|
| 일본어 | 風牌 (カゼハイ) |
| 로마자 | kazehai |
| 한국어 | 풍패 |

## 정의

풍패는 동(東), 남(南), 서(西), 북(北) 4종류의 자패를 말한다.

## 상세 설명

각 풍패는 4장씩 총 16장이다. 자신의 자풍(自風)이나 장풍(場風)에 해당하는 풍패의 커쯔를 만들면 역패(yakuhai)가 성립한다. 소사희(shousuushii)는 풍패 3종의 커쯔+1종의 또이쯔, 대사희(daisuushii)는 풍패 4종 모두 커쯔로 구성하는 역만이다.

## 사용 예시

- "동장의 동가는 동풍 커쯔로 역패 + 연풍패 더블동이 된다."
- "풍패는 자풍·장풍 여부를 항상 확인해야 한다."
