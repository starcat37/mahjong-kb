# 삼색동각 (三色同刻)

| 항목 | 내용 |
|------|------|
| 일본어 | 三色同刻 |
| 로마자 | sanshoku doukou |
| 한국어 | 삼색동각 |
| 판수 | 2판 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

삼색동각은 만수, 통수, 삭수 세 종류에서 같은 숫자의 커쯔를 각각 1개씩 만드는 역이다. 삼색동순의 커쯔 버전으로, 후로해도 쿠이사가리 없이 2판이 유지된다.

## 성립 조건

1. 만수, 통수, 삭수에서 동일한 숫자의 커쯔(또는 깡쯔)를 각 1개씩 총 3개 보유해야 한다.
2. 나머지 1개의 몸통과 머리는 자유롭게 구성할 수 있다.
3. 후로(퐁) 가능하며 쿠이사가리가 적용되지 않는다.

## 예시

### 기본 예시

만수 555, 통수 555, 삭수 555, 만수 123, 동동 (머리)
→ 만·통·삭 모두에서 5의 커쯔가 있으므로 삼색동각 성립

### 주의할 패턴

- 만수 555, 통수 555, 삭수 333처럼 숫자가 다르면 성립하지 않는다.
- 같은 숫자의 커쯔 3개가 필요하므로 출현 빈도가 매우 낮은 역이다.

## 전략적 활용

- 같은 숫자의 패를 세 종류 모두에서 3장씩 모아야 하므로 의도적으로 노리기 어렵다.
- 또이또이와 자연스럽게 복합되는 경우가 많다 (커쯔 3개 + 커쯔 1개 구성).
- 퐁을 적극 활용하여 커쯔를 모으는 전략이 필요하다.
- 출현 빈도가 낮으므로 배패에서 같은 숫자의 또이쯔가 여러 종류에 걸쳐 있을 때만 고려한다.
- 악명 높은 역 중 하나로, 일부 역만보다도 나오기 힘들면서 판수는 2판에 불과하다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 삼색동각이 포함되어 있을 확률은 약 **0.037%**로, 국사무쌍(0.036%)이나 대삼원(0.031%)과 비슷한 수준이다. 역만급 난이도에 2판이라는 점에서 효율이 매우 낮은 역이다.
