# 텐호 (天和)

| 항목 | 내용 |
|------|------|
| 일본어 | 天和 |
| 로마자 | tenhou |
| 한국어 | 텐호 |
| 판수 | 역만 |
| 후로 가능 | 불가 (멘젠만) |
| 쿠이사가리 | 해당 없음 |

## 개요

텐호는 친(동가, 딜러)이 배패 14장을 받은 시점에서 이미 화료 형태가 완성되어 있는 역만이다. 완전히 운에 의존하는 우연역으로, 배패를 받자마자 즉시 화료를 선언한다.

## 성립 조건

1. 친(동가)이어야 한다.
2. 배패로 받은 14장이 그대로 화료 형태를 이루어야 한다.
3. 쯔모를 하기 전, 배패 시점에서 바로 화료를 선언한다.
4. 어떤 화료 형태든 상관없다(핑후, 또이또이, 치또이쯔 등).

## 예시

### 기본 예시

```
배패: 1m2m3m 4p5p6p 7s8s9s 東東東 白白
```
친이 배패 14장을 받은 순간 화료 형태 완성 → 텐호 선언

```
배패: 1m1m 2p2p 3s3s 東東 南南 白白 中中
```
치또이쯔 형태로 배패 완성 → 텐호 선언

### 주의할 패턴

- 자(코)가 배패에서 화료 형태가 되어도 텐호가 아닌 치호(지화)이다.
- 친이라도 배패 후 1장이라도 쯔모한 뒤 화료하면 텐호가 성립하지 않는다.
- 확률이 극히 낮아 실전에서 거의 볼 수 없다(약 33만 분의 1).

## 전략적 활용

- 순수한 운의 역이므로 전략적으로 노릴 수 있는 역이 아니다.
- 배패를 받은 직후 화료 형태를 빠르게 확인하는 습관이 중요하다.
- 온라인 마작에서는 자동으로 텐호를 감지해주는 경우가 많다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 텐호가 포함되어 있을 확률은 약 **0.00034%**이다.
