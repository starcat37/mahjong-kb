# 쇼산겐 (小三元)

| 항목 | 내용 |
|------|------|
| 일본어 | 小三元 |
| 로마자 | shousangen |
| 한국어 | 쇼산겐 |
| 판수 | 2판 (실질 4판: 역패 2판 자동 포함) |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

쇼산겐(소삼원)은 삼원패(백, 발, 중) 중 2종류를 커쯔(또는 깡쯔)로, 나머지 1종류를 머리로 구성하는 역이다. 쇼산겐 자체는 2판이지만, 삼원패 커쯔 2개가 각각 역패로 1판씩 부여되므로 실질적으로 최소 4판이 확정된다.

## 성립 조건

1. 백, 발, 중 중 2종류를 커쯔(또는 깡쯔)로 보유해야 한다.
2. 나머지 1종류의 삼원패를 머리(또이쯔)로 가져야 한다.
3. 나머지 2개의 몸통은 자유롭게 구성할 수 있다.
4. 후로(퐁) 가능하며 쿠이사가리가 적용되지 않는다.

## 예시

### 기본 예시

백백백, 발발발, 만수 234, 삭수 678, 중중 (머리)
→ 삼원패 중 백·발이 커쯔, 중이 머리이므로 쇼산겐 성립
→ 쇼산겐(2판) + 역패 백(1판) + 역패 발(1판) = 4판

### 주의할 패턴

- 삼원패 3종류 모두 커쯔로 만들면 쇼산겐이 아닌 대삼원(역만)이 된다.
- 삼원패 커쯔가 1개뿐이면 쇼산겐이 성립하지 않으며 역패만 적용된다.

## 전략적 활용

- 실질 4판이 확정되므로 후로해도 충분한 점수가 나온다.
- 삼원패 2종류의 또이쯔가 배패에 있으면 적극적으로 퐁을 노린다.
- 호닛쯔(혼일색)와 복합하면 후로 상태에서도 6판 이상으로 높은 점수를 기대할 수 있다.
- 대삼원을 노리다가 삼원패 1종류가 부족할 때 쇼산겐으로 전환하는 경우도 있다.
