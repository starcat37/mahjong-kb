# 샨텐 (向聴 / shanten)

| 항목 | 내용 |
|------|------|
| 일본어 | 向聴 |
| 로마자 | shanten |
| 한국어 | 샨텐 |

## 정의

텐파이까지 필요한 유효패의 수. e.g.) 이샨텐(1), 량샹텐(2).
