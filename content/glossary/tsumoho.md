# 쯔모호 (ツモ和 / tsumoho)

| 항목 | 내용 |
|------|------|
| 일본어 | ツモ和 |
| 로마자 | tsumoho |
| 한국어 | 쯔모호 |

## 정의

자신의 차례에 패산에서 패를 가져와서 화료하는 것. 멘젠일 경우 '멘젠친쯔모호'라는 1판역이 성립하며, 멘젠 여부와 상관없이 부수가 2부 붙는다. 줄여서 '쯔모'라고 부르는 경우가 대부분이다.
