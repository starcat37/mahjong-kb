# 왕패 (王牌 / wanpai)

| 항목 | 내용 |
|------|------|
| 일본어 | 王牌 (ワンパイ) |
| 로마자 | wanpai |
| 한국어 | 왕패 |

## 정의

왕패(王牌)는 패산 끝에 분리해놓는 14장으로, 도라 표시패와 영상패를 포함한다.

## 상세 설명

왕패는 항상 14장을 유지하며 일반 쯔모에 사용하지 않는다. 왕패 위쪽의 3번째 패가 도라 표시패. 깡을 하면 왕패의 영상패에서 1장을 가져오고, 패산 끝에서 1장을 왕패로 보충한다.

## 사용 예시

- 왕패의 도라 표시패를 확인한다
- 깡 후 왕패에서 영상패를 쯔모한다
