# 린샨카이호 (嶺上開花)

| 항목 | 내용 |
|------|------|
| 일본어 | 嶺上開花 |
| 로마자 | rinshan kaihou |
| 한국어 | 영상개화 |
| 판수 | 1판 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

린샨카이호는 깡을 선언한 뒤 영상패(깡 후 보충하는 패)로 쯔모 화료하는 역이다. 깡 자체가 빈번하지 않은 데다 영상패가 당첨패여야 하므로 발생 빈도는 낮지만, 깡을 적극적으로 활용하는 플레이에서 가끔 볼 수 있다.

## 성립 조건

1. 깡(암깡, 대명깡, 가깡 모두 해당)을 선언해야 한다.
2. 깡 후 가져오는 영상패로 쯔모 화료해야 한다.

## 예시

### 기본 예시

만수 5555로 암깡 선언 후 영상패로 통수 3을 가져왔는데, 이것이 대기패와 일치하여 쯔모 화료 — 린샨카이호 성립.

### 주의할 패턴

- 영상패를 가져온 뒤 화료하지 못하고 버린 패로 다른 사람이 론하는 것은 린샨카이호가 아니다.
- 깡 후 영상패로 화료하면 하이테이(해저로월)와는 복합하지 않는다. 영상패는 해저패로 취급하지 않기 때문이다.

## 전략적 활용

- 깡을 할 때 부수적으로 노릴 수 있는 역이다. 텐파이 상태에서 깡이 가능하면 영상패로의 화료 기회가 추가된다.
- 도라가 늘어나는 깡의 이점과 함께 린샨카이호 가능성까지 있으므로, 텐파이 상태에서의 깡은 공격적 선택이 된다.
- 초보자는 "깡 후 가져온 패로 쯔모하면 1판 추가"라고 기억하면 된다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 영상개화가 포함되어 있을 확률은 약 **0.30%**이다. 우연역답게 출현 빈도가 낮다.
