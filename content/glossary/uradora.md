# 뒷도라 (裏ドラ / uradora)

| 항목 | 내용 |
|------|------|
| 일본어 | 裏ドラ (うらドラ) |
| 로마자 | uradora |
| 한국어 | 뒷도라 |

## 정의

뒷도라(裏ドラ)는 리치로 화료 성공 시 도라 표시패 아래에 있는 패로 결정되는 추가 도라.

## 상세 설명

리치 선언 후 화료해야만 확인할 수 있다. 뒷도라 표시패도 일반 도라와 같은 방식으로 다음 패가 뒷도라가 된다. 운 요소가 크지만 고득점을 노릴 수 있어 리치의 매력을 높인다.

## 사용 예시

- 리치 쯔모에 뒷도라가 2개 실려 하네만!
- 뒷도라 확인은 리치 화료 시에만 가능
