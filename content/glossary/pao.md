# 파오 (責任払い / pao)

| 항목 | 내용 |
|------|------|
| 일본어 | 責任払い (せきにんばらい) |
| 로마자 | pao |
| 한국어 | 파오 |

## 정의

파오(責任払い)는 특정 역만을 확정시킨 플레이어가 점수 지불 책임을 지는 규칙.

## 상세 설명

대삼원의 3번째 삼원패를 울게 하거나, 대사희의 4번째 풍패를 울게 한 플레이어에게 파오가 적용된다. 파오 적용 시 쯔모 화료이면 파오 플레이어가 전액 지불, 론 화료이면 방총자와 파오 플레이어가 반반 부담한다. 사깡쯔도 파오 적용 대상인 룰이 있다.

## 사용 예시

- "대삼원 3번째 발을 울게 해준 플레이어에게 파오 적용"
- "파오 + 쯔모이면 파오 플레이어가 역만 전액 부담"
