# 도중유국 (途中流局 / chuukan ryuukyoku)

| 항목 | 내용 |
|------|------|
| 일본어 | 途中流局 |
| 로마자 | chuukan ryuukyoku |
| 한국어 | 도중유국 |

## 정의

특별한 조건에 의해 국 진행 도중 유국 처리가 되는 것. 이 경우 연장봉을 1개 추가한 채로 국을 다시 시작한다. e.g.) 사풍연타, 사깡유국 등.
