# 슌쯔 (順子 / shuntsu)

| 항목 | 내용 |
|------|------|
| 일본어 | 順子 (シュンツ) |
| 로마자 | shuntsu |
| 한국어 | 슌쯔 |

## 정의

슌쯔(順子)는 같은 종류의 수패 3장이 연속된 숫자로 이루어진 멘쯔이다.

## 상세 설명

예를 들어 2만·3만·4만이나 5통·6통·7통처럼 연속된 3장으로 구성된다. 자패로는 슌쯔를 만들 수 없다. 다른 플레이어의 타패를 치(チー)하여 만들 수도 있다. 핑후(pinfu)는 멘젠에서 슌쯔만으로 구성해야 한다.

## 사용 예시

- "1-2-3 만수패로 슌쯔 하나를 완성했다."
- "치를 하면 슌쯔가 되지만 멘젠이 깨진다."
