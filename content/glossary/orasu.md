# 오라스 (オーラス / oorasu)

| 항목 | 내용 |
|------|------|
| 일본어 | オーラス |
| 로마자 | oorasu |
| 한국어 | 오라스 |

## 정의

동풍전의 동 4국과 반장전의 남 4국을 의미하는 말. 어원은 영어의 'All last'.
