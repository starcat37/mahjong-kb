# 나시나시 (ナシナシ / nashinashi)

| 항목 | 내용 |
|------|------|
| 일본어 | ナシナシ |
| 로마자 | nashinashi |
| 한국어 | 나시나시 |

## 정의

멘젠을 깬 탕야오(쿠이탕)와 아토즈케를 인정하지 않는 룰을 지칭하는 말. 반의어: 아리아리.
