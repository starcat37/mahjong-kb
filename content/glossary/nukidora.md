# 누키도라 (抜きドラ / nukidora)

| 항목 | 내용 |
|------|------|
| 일본어 | 抜きドラ |
| 로마자 | nukidora |
| 한국어 | 누키도라 |

## 정의

빼놓는다고 선언하는 것으로 도라가 되는 특수한 패. 주로 3인 마작에서 채택하며 이 경우 대체로 자패의 北을 누키도라로 쓴다.
