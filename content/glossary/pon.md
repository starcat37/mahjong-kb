# 퐁 (ポン / pon)

| 항목 | 내용 |
|------|------|
| 일본어 | ポン |
| 로마자 | pon |
| 한국어 | 퐁 |

## 정의
퐁(ポン)은 어떤 플레이어가 버린 패를 가져와 커쯔를 만드는 울기.

## 상세 설명
치와 달리 모든 플레이어의 타패에 대해 퐁 가능. 치보다 우선순위가 높지만 론보다는 낮다. 퐁을 하면 밍커가 되어 멘젠이 깨진다.

## 사용 예시
- "대면이 버린 발을 퐁하여 역패 확정"
- "퐁은 아무 플레이어의 패든 가능하다"
