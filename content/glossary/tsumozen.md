# 쯔모센 (ツモ専 / tsumo sen)

| 항목 | 내용 |
|------|------|
| 일본어 | ツモ専 |
| 로마자 | tsumo sen |
| 한국어 | 쯔모센 |

## 정의

텐파이했지만 쯔모로만 화료 가능한 상태를 뜻하는 말. e.g.) 후리텐 텐파이, 산안커의 아토즈케.

- [[tenpai]] — 텐파이
