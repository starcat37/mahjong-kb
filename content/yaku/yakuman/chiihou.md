# 치호 (地和)

| 항목 | 내용 |
|------|------|
| 일본어 | 地和 (ちほう) |
| 로마자 | chiihou |
| 한국어 | 치호 / 지화 |

| 쿠이사가리 | 없음 |

## 개요

치호(地和, 지화)는 자(子, 친이 아닌 플레이어)가 첫 번째 쯔모로 화료하는 역만이다. 천화(天和)가 친 전용이라면, 치호는 자 전용의 첫 쯔모 화료 역만이다.

## 성립 조건

1. 자신이 자(子)일 것 (친이 아닌 남가, 서가, 북가)
2. 자신의 첫 쯔모에서 화료할 것
3. 자신의 쯔모 이전에 어느 누구도 울기(치/퐁/깡)를 하지 않았을 것
4. 멘젠 상태일 것

## 예시

### 기본 예시

배패 13장이 텐파이 상태이고, 첫 쯔모패가 대기패인 경우:

- 배패: 만수 123, 통수 456, 삭수 789, 동동동, 백
- 첫 쯔모: 백 → 치호 성립

### 주의할 패턴

- 친의 타패 후 첫 쯔모이므로 성립 가능하지만, 그 사이 다른 자가 친의 버림패를 울면 치호 불성립
- 동가(친)라면 치호가 아니라 텐호가 됨
- 자신의 차례 전에 타가가 울기를 한 경우 불성립

## 전략적 활용

- 치호는 완전히 운에 의존하는 역만으로, 의도적으로 노릴 수 없다
- 배패가 텐파이 상태라면 자동으로 치호를 기대할 수 있으며, 자신의 차례까지 아무도 울지 않기를 바라는 수밖에 없다
- 텐호와 마찬가지로 일생에 한 번 나올까 말까 한 극히 희귀한 역만이다
