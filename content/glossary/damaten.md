# 다마텐 (黙聴 / damaten)

| 항목 | 내용 |
|------|------|
| 일본어 | 黙聴 |
| 로마자 | damaten |
| 한국어 | 다마텐 |

## 정의

멘젠으로 텐파이한 상태로 리치를 걸지 않고 기다리는 행위. 어원은 '말없이 텐파이(黙ってテンパイ)'. 동의어: 야미텐.

- [[menzen]] — 멘젠
