# 라스회피 (ラス回避 / rasu kaihi)

| 항목 | 내용 |
|------|------|
| 일본어 | ラス回避 |
| 로마자 | rasu kaihi |
| 한국어 | 라스회피 |

## 정의

적극적으로 4위를 피하는 것. 주로 오라스에서 3위로 순위를 확정지어 그 시합을 끝내는 것을 지칭한다.
