# 량페코 (二盃口)

| 항목 | 내용 |
|------|------|
| 일본어 | 二盃口 |
| 로마자 | ryanpeikou |
| 한국어 | 량페코 |
| 판수 | 3판 |
| 후로 가능 | 불가 (멘젠만) |
| 쿠이사가리 | 해당 없음 |

## 개요

량페코는 이페코(동일 슌쯔 2개)가 2쌍 존재하는 역이다. 멘젠 전용 3판역으로, 치또이쯔와 동시에 성립할 수 없다. 실전에서 완성 난이도가 높아 희소가치가 있는 역이다.

## 성립 조건

1. 멘젠(문전) 상태여야 한다.
2. 동일한 슌쯔 2개로 이루어진 조합이 2쌍 존재해야 한다.
3. 머리(작두) 1조가 필요하다.
4. 치또이쯔(七対子)와는 복합되지 않는다.

## 예시

### 기본 예시

```
1m1m2m2m3m3m 5p5p6p6p7p7p 9s9s
```
만수 123 이페코 + 통수 567 이페코 + 삭수 99 머리

```
2s2s3s3s4s4s 7m7m8m8m9m9m 1p1p
```
삭수 234 이페코 + 만수 789 이페코 + 통수 11 머리

### 주의할 패턴

- 패 구성이 치또이쯔와 동일하게 보일 수 있으나, 량페코는 슌쯔 해석이 우선된다. 치또이쯔와는 중복되지 않는다.
- 후로(치, 퐁, 깡)를 하면 성립하지 않는다.
- 111122223333 같은 형태도 112233 x 2로 해석하여 량페코로 인정된다.

## 전략적 활용

- 이페코가 하나 완성된 상태에서 나머지 패가 또 다른 이페코 형태에 가까울 때 노려볼 수 있다.
- 멘젠 유지가 필수이므로, 리치와 복합하면 최소 4판 이상을 기대할 수 있다.
- 치또이쯔와 동일 패 구성이 되는 경우, 량페코 쪽이 판수가 높으므로 량페코로 해석한다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 량페코가 포함되어 있을 확률은 약 **0.043%**로, 스안커(0.042%)와 비슷한 수준이다. 3판역이지만 역만급 난이도를 지닌다.
