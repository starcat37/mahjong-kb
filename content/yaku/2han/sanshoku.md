# 삼색동순 (三色同順)

| 항목 | 내용 |
|------|------|
| 일본어 | 三色同順 |
| 로마자 | sanshoku doujun |
| 한국어 | 삼색동순 |
| 판수 | 멘젠 2판 / 후로 1판 |
| 후로 가능 | 가능 (쿠이사가리 적용) |
| 쿠이사가리 | 있음 (후로 시 1판) |

## 개요

삼색동순은 만수, 통수, 삭수 세 종류에서 같은 숫자 조합의 슌쯔를 각각 1개씩 만드는 역이다. 멘젠 시 2판이지만 후로하면 쿠이사가리가 적용되어 1판으로 감소한다.

## 성립 조건

1. 만수, 통수, 삭수에서 동일한 숫자 구성의 슌쯔를 각 1개씩 총 3개 보유해야 한다.
2. 나머지 1개 몸통과 머리는 자유롭게 구성할 수 있다.
3. 후로(치, 퐁) 가능하지만 쿠이사가리로 1판이 된다.

## 예시

### 기본 예시

만수 234, 통수 234, 삭수 234, 만수 678, 삭수 55 (머리)
→ 234 슌쯔가 세 종류 모두에 있으므로 삼색동순 성립

### 주의할 패턴

- 만수 123, 통수 123, 삭수 234처럼 숫자 구성이 하나라도 다르면 성립하지 않는다.
- 치로 슌쯔 하나를 울었을 경우 후로 판정이 되어 1판으로 감소한다.

## 전략적 활용

- 핑후와 복합하기 쉬워 멘젠으로 완성하면 높은 점수를 기대할 수 있다.
- 같은 숫자대의 패를 세 종류 모두 모아야 하므로 중반 이후에는 완성이 어려워진다.
- 후로 시 1판으로 감소하기 때문에 가능하면 멘젠으로 노리는 것이 유리하다.
- 잇쯔(일기통관)와는 동시 성립이 불가능하므로 어느 쪽을 노릴지 판단이 필요하다.
