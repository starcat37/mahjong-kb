# 데바사이 (出場所最高 / debasai)

| 항목 | 내용 |
|------|------|
| 일본어 | 出場所最高 |
| 로마자 | debasai |
| 한국어 | 데바사이 |

## 정의

점수 상황상 직격하고 싶은 상대에게 론 화료하는 것. '데바쇼 사이코(出場所最高)'의 약어. 화료한 사람이 상대에게 직접 '데바사이' 운운하는 건 매너 위반으로 여겨진다.
