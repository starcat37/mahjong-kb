# 노텐 (ノーテン / noten)

| 항목 | 내용 |
|------|------|
| 일본어 | ノーテン |
| 로마자 | noten |
| 한국어 | 노텐 |

## 정의

황패유국시 텐파이하지 못한 경우를 뜻한다. 노텐인 사람은 손패를 덮는 것으로 노텐임을 밝히고, 텐파이한 사람에게 벌금을 지불한다.
