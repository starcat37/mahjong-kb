# 서입 (西入 / nishi iri)

| 항목 | 내용 |
|------|------|
| 일본어 | 西入 |
| 로마자 | nishi iri |
| 한국어 | 서입 |

## 정의

반장전의 오라스가 끝난 시점에서 반환점을 넘긴 사람이 없을 경우 진행되는 이후의 국. 동풍전도 같은 상황이 벌어질 수 있는데, 이때는 남입이 된다.
