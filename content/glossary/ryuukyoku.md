# 유국 (流局 / ryuukyoku)

| 항목 | 내용 |
|------|------|
| 일본어 | 流局 (りゅうきょく) |
| 로마자 | ryuukyoku |
| 한국어 | 유국 |

## 정의

유국(流局)은 아무도 화료하지 못한 채 패산이 소진되어 국이 종료되는 상황.

## 상세 설명

유국 시 텐파이인 플레이어는 노텐(不聴) 플레이어로부터 점수를 받는다(텐파이 1명이면 3000점, 2명이면 각 1500점, 3명이면 각 1000점). 특수 유국으로 구종구패, 사풍연타, 사가리치, 삼가화료, 사깡유국 등이 있다.

## 사용 예시

- "유국 시 텐파이 여부에 따라 점수 이동이 있다"
- "특수 유국은 즉시 국이 종료된다"
