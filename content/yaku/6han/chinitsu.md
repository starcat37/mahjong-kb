# 친잇쯔 (清一色)

| 항목 | 내용 |
|------|------|
| 일본어 | 清一色 |
| 로마자 | chinitsu |
| 한국어 | 친잇쯔 |
| 판수 | 멘젠 6판 / 후로 5판 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 있음 (후로 시 5판) |

## 개요

친잇쯔는 만수, 통수, 삭수 중 한 종류의 수패만으로 손패 전체를 구성하는 역이다. 멘젠 6판, 후로 5판으로 역만을 제외하면 단일 역 중 최고 판수를 자랑한다. 호닛쯔의 상위 호환에 해당한다.

## 성립 조건

1. 수패는 만수, 통수, 삭수 중 한 종류만 사용해야 한다.
2. 자패(풍패, 삼원패)는 사용할 수 없다.
3. 후로 시 쿠이사가리가 적용되어 5판으로 감소한다.

## 예시

### 기본 예시

```
1m1m1m 2m3m4m 5m6m7m 8m9m9m 9m9m (대기: 7m 또는 기타)
```
만수만으로 구성된 패

```
1s2s3s 3s4s5s 6s7s8s 8s8s8s 2s2s
```
삭수만으로 구성: 삭수 123 + 345 + 678 + 888 + 22 머리

### 주의할 패턴

- 자패가 하나라도 포함되면 호닛쯔(혼일색)로 격하된다.
- 한 종류 수패만 사용하므로 대기 패턴이 복잡해질 수 있어, 다면대기 판정에 주의가 필요하다.

## 전략적 활용

- 배패에 한 종류 수패가 7~8장 이상 집중되어 있을 때 적극적으로 노릴 수 있다.
- 후로해도 5판이므로 만관급 이상의 높은 점수를 기대할 수 있다.
- 한 색만 모으므로 상대에게 쉽게 읽히는 단점이 있어, 멘젠으로 리치를 걸면 하네만 이상이 확정된다.
- 호닛쯔를 목표로 진행하다가, 자패를 버리고 친잇쯔로 전환하는 패턴이 흔하다.
- 다면대기가 되기 쉬워 텐파이 시 유리한 대기를 만들 수 있다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 청일색이 포함되어 있을 확률은 약 **0.77%**이다.
