# 슬라이드 (スライド / slide)

| 항목 | 내용 |
|------|------|
| 일본어 | スライド |
| 로마자 | slide |
| 한국어 | 슬라이드 |

## 정의

슌쯔의 가장자리에 붙는 패를 쯔모했을 때 반대쪽 패를 버려 슌쯔의 숫자를 바꾸는 행위. e.g.) 345만을 가지고 있을 때 6만을 쯔모해서 3만을 버림.
