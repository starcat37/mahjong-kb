# 혼로토 (混老頭)

| 항목 | 내용 |
|------|------|
| 일본어 | 混老頭 |
| 로마자 | honroutou |
| 한국어 | 혼로토 |
| 판수 | 2판 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

혼로토(혼노두)는 노두패(수패 1, 9)와 자패만으로 손패 전체를 구성하는 역이다. 수패 2~8이 하나도 포함되지 않아야 하며, 구조적으로 슌쯔를 만들 수 없으므로 또이또이 또는 치또이쯔와 반드시 복합된다.

## 성립 조건

1. 모든 몸통과 머리가 노두패(수패 1, 9) 또는 자패로만 구성되어야 한다.
2. 수패 2~8이 하나라도 포함되면 성립하지 않는다.
3. 슌쯔를 만들 수 없으므로 커쯔/깡쯔 4개 + 머리(또이또이형) 또는 또이쯔 7개(치또이쯔형)로 구성된다.
4. 후로(퐁) 가능하며 쿠이사가리가 적용되지 않는다.

## 예시

### 기본 예시

만수 111, 통수 999, 삭수 111, 동동동, 북북 (머리)
→ 모든 패가 수패 1, 9 또는 자패이므로 혼로토 성립
→ 혼로토(2판) + 또이또이(2판) = 4판

### 주의할 패턴

- 치또이쯔형: 만수 11, 통수 11, 삭수 99, 동동, 서서, 백백, 중중
  → 혼로토(2판) + 치또이쯔(2판) = 4판
- 자패 없이 수패 1, 9만으로 구성하면 혼로토가 아닌 친로토(청노두, 역만)가 된다.

## 전략적 활용

- 또이또이와 복합하면 최소 4판이 확정되어 안정적인 점수를 기대할 수 있다.
- 역패(자풍패, 장풍패, 삼원패)와 추가 복합하면 판수가 더욱 올라간다.
- 배패에서 노두패와 자패가 많을 때 퐁을 적극 활용하여 노린다.
- 찬타와 비교하면 슌쯔가 없는 대신 쿠이사가리가 없어 후로해도 2판이 유지되는 장점이 있다.
