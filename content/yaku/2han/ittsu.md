# 잇쯔 (一気通貫)

| 항목 | 내용 |
|------|------|
| 일본어 | 一気通貫 |
| 로마자 | ikkitsuukan |
| 한국어 | 일기통관 |
| 판수 | 멘젠 2판 / 후로 1판 |
| 후로 가능 | 가능 (쿠이사가리 적용) |
| 쿠이사가리 | 있음 (후로 시 1판) |

## 개요

잇쯔(일기통관)는 한 종류의 수패로 123, 456, 789 세 개의 슌쯔를 모두 완성하는 역이다. 1부터 9까지를 한 번에 관통한다는 의미로, 멘젠 시 2판, 후로 시 쿠이사가리가 적용되어 1판이 된다.

## 성립 조건

1. 만수, 통수, 삭수 중 한 종류에서 123, 456, 789 슌쯔를 모두 갖추어야 한다.
2. 나머지 1개 몸통과 머리는 자유롭게 구성할 수 있다.
3. 후로(치) 가능하지만 쿠이사가리로 1판이 된다.

## 예시

### 기본 예시

만수 123, 만수 456, 만수 789, 통수 567, 삭수 33 (머리)
→ 만수로 123·456·789 슌쯔를 모두 갖추었으므로 잇쯔 성립

### 주의할 패턴

- 만수 123, 통수 456, 삭수 789처럼 서로 다른 종류에 걸쳐 있으면 성립하지 않는다. 반드시 같은 종류여야 한다.
- 만수 123, 만수 456까지 있고 만수 789 대신 만수 78만 있는 텐파이 상태에서 만수 9를 기다리는 경우, 화료하면 잇쯔가 성립한다.

## 전략적 활용

- 한 종류의 수패를 많이 모아야 하므로 배패에서 특정 수패가 편중되어 있을 때 노리기 좋다.
- 핑후와 복합하면 멘젠에서 4판이 되어 높은 득점을 기대할 수 있다.
- 후로 시 1판으로 감소하므로 다른 역과의 복합 없이 후로만으로는 점수가 낮다.
- 삼색동순과는 같은 슌쯔 기반 역이지만 동시에 성립할 수 없으므로 택일해야 한다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 일기통관이 포함되어 있을 확률은 약 **1.55%**이다.
