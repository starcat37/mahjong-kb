# 또이또이 (対々和)

| 항목 | 내용 |
|------|------|
| 일본어 | 対々和 |
| 로마자 | toitoihou |
| 한국어 | 또이또이 |
| 판수 | 2판 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

또이또이는 4개의 커쯔(또는 깡쯔)와 1개의 머리로 손패를 구성하는 역이다. 슌쯔 없이 모두 커쯔/깡쯔로만 이루어져야 하며, 후로해도 판수 감소 없이 2판이 유지된다.

## 성립 조건

1. 4개의 몸통이 모두 커쯔(같은 패 3장) 또는 깡쯔(같은 패 4장)여야 한다.
2. 슌쯔(연속된 3장)가 하나라도 있으면 성립하지 않는다.
3. 1개의 머리(또이쯔)가 필요하다.
4. 후로(퐁) 가능하며 쿠이사가리 없이 2판 유지.

## 예시

### 기본 예시

만수 111, 통수 333, 삭수 777, 동동동, 발발 (머리)
→ 4개의 커쯔 + 1개의 머리로 구성되었으므로 또이또이 성립

### 주의할 패턴

- 멘젠에서 또이또이를 완성하면 안커 4개가 되어 스안커(역만)가 성립할 수 있다. 단, 론으로 화료하면 마지막 커쯔가 밍커가 되므로 산안코 + 또이또이가 된다.
- 또이또이 텐파이에서 퐁 3회 후 나머지가 커쯔 + 머리 형태인지 확인해야 한다.

## 전략적 활용

- 배패에서 또이쯔(2장 쌍)가 많을 때 퐁을 적극 활용하여 노린다.
- 역패와 복합하기 쉬워 퐁을 통해 빠르게 화료를 목표로 할 수 있다.
- 혼로토와 복합하면 또이또이(2판) + 혼로토(2판) = 4판이 된다.
- 후로가 많아지면 손패가 적어져 방어가 어려우므로 공격적인 국면에서 활용한다.
