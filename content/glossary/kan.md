# 깡 (カン / kan)

| 항목 | 내용 |
|------|------|
| 일본어 | カン |
| 로마자 | kan |
| 한국어 | 깡 |

## 정의
깡(カン)은 같은 패 4장을 모아 깡쯔를 선언하는 행위.

## 상세 설명
안깡·밍깡·가깡 세 종류가 있다. 깡 선언 후 영상패에서 1장을 가져오고 깡도라 표시패가 추가된다. 한 국에서 깡은 최대 4회까지 가능(5번째 깡 시 유국).

## 사용 예시
- "같은 패 4장을 모아 깡 선언"
- "깡하면 도라가 늘어 위험할 수 있다"
