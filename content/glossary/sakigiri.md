# 사키기리 (先切り / sakigiri)

| 항목 | 내용 |
|------|------|
| 일본어 | 先切り |
| 로마자 | sakigiri |
| 한국어 | 사키기리 |

## 정의

두 종류의 패를 버릴 때, 발전 가능성이 있는 패를 굳이 먼저 버리고 상대적으로 안전한 패를 잠시 손에 남기는 행위.
