# 텐파이 (聴牌 / tenpai)

| 항목 | 내용 |
|------|------|
| 일본어 | 聴牌 (テンパイ) |
| 로마자 | tenpai |
| 한국어 | 텐파이 |

## 정의
텐파이(聴牌)는 1장만 더 오면 화료할 수 있는 상태.

## 상세 설명
텐파이 상태에서 리치를 선언하거나, 론·쯔모로 화료를 노린다. 유국 시 텐파이인 플레이어는 노텐(불청) 플레이어로부터 점수를 받는다. 대기 형태에 따라 양면·간짱·변짱·단기·샤보 등으로 나뉜다.

## 사용 예시
- "리치를 걸려면 먼저 텐파이여야 한다"
- "유국 시 텐파이 3명이면 노텐 벌금은 1명이 3000점"
