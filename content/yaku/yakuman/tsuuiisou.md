# 쯔이소 (字一色)

| 항목 | 내용 |
|------|------|
| 일본어 | 字一色 |
| 로마자 | tsuuiisou |
| 한국어 | 쯔이소 |
| 판수 | 역만 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

쯔이소는 손패 전체를 자패(풍패 4종 + 삼원패 3종)만으로 구성하는 역만이다. 수패를 일절 사용하지 않으며, 커쯔 형태로만 구성된다. 후로해도 역만이 유지된다.

## 성립 조건

1. 손패의 모든 몸통과 머리가 자패(東, 南, 西, 北, 白, 發, 中)로만 이루어져야 한다.
2. 수패(만수, 통수, 삭수)는 사용할 수 없다.
3. 자패는 슌쯔를 만들 수 없으므로 커쯔(또는 깡쯔) 형태만 가능하다.
4. 치또이쯔 형태의 쯔이소도 가능하다(룰에 따라 다름).

## 예시

### 기본 예시

```
東東東 南南南 西西西 白白白 中中
```
동동동 + 남남남 + 서서서 + 백백백 + 중중 머리

```
北北北 白白白 發發發 中中中 東東
```
북북북 + 백백백 + 발발발 + 중중중 + 동동 머리

### 주의할 패턴

- 대삼원, 쇼스시, 다이스시와 복합될 수 있다.
- 수패가 하나라도 포함되면 성립하지 않는다.
- 또이또이(대대화)가 자동으로 포함되지만, 역만이므로 추가 판수 계산은 하지 않는다.

## 전략적 활용

- 배패에 자패가 대량으로 몰려 있을 때 노릴 수 있다.
- 자패 7종 중 최소 5종이 필요하므로 희소성이 매우 높다.
- 후로를 적극 활용하면 속도를 올릴 수 있지만, 상대에게 역만이 읽히기 쉽다.
- 호닛쯔를 노리다가 수패를 포기하고 쯔이소로 전환하는 경우는 드물다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 쯔이소가 포함되어 있을 확률은 약 **0.0053%**이다. 치또이쯔(2-2-2-2-2-2-2) 형태로도 인정되므로 커쯔 형태 외에도 가능성이 있다.
