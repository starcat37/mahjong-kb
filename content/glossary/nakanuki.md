# 나카누키 (中抜き / nakanuki)

| 항목 | 내용 |
|------|------|
| 일본어 | 中抜き |
| 로마자 | nakanuki |
| 한국어 | 나카누키 |

## 정의

방총을 피하기 위해, 자신이 가지고 있는 슌쯔의 가운데 부분 패를 빼서 버리는 행위. e.g.) 345만의 슌쯔를 가지고 있을 때, 리치자의 현물인 4만을 버린 경우.
