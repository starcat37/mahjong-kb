# 나카부쿠레 (中膨れ / nakabukure)

| 항목 | 내용 |
|------|------|
| 일본어 | 中膨れ |
| 로마자 | nakabukure |
| 한국어 | 나카부쿠레 |

## 정의

슌쯔 + 해당 슌쯔의 가운데 수패가 하나(혹은 그 이상) 더 있는 형태를 지칭하는 말. e.g.) 3445, 23334, 566667 등.
