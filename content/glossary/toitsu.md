# 또이쯔 (対子 / toitsu)

| 항목 | 내용 |
|------|------|
| 일본어 | 対子 (トイツ) |
| 로마자 | toitsu |
| 한국어 | 또이쯔 |

## 정의

또이쯔(対子)는 동일한 패 2장의 쌍을 말한다.

## 상세 설명

일반 화료에서 머리(아타마) 역할을 하며, 치또이쯔(칠대자)는 또이쯔 7쌍으로 화료한다. 또이쯔가 2개 있으면 샤보(쌍봉) 대기가 되며, 이 경우 어느 쪽이 커쯔가 되든 화료가 가능하다.

## 사용 예시

- "머리용 또이쯔가 없으면 텐파이가 되지 않는다."
- "치또이쯔는 또이쯔 7개를 모아야 한다."
