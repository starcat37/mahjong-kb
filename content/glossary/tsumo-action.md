# 쯔모(행동) (ツモ / tsumo)

| 항목 | 내용 |
|------|------|
| 일본어 | ツモ |
| 로마자 | tsumo |
| 한국어 | 쯔모 |

## 정의

쯔모(ツモ)는 자신의 차례에 패산에서 패 1장을 가져오는 행동.

## 상세 설명

매 턴마다 플레이어는 패산에서 1장을 쯔모하고, 불필요한 패 1장을 버린다(타패). 쯔모한 패로 화료하면 '쯔모 화료'가 되며, 멘젠쯔모는 역으로 인정된다(1판). 역 이름의 쯔모(멘젠쯔모)와 행동으로서의 쯔모를 구분할 것.

## 사용 예시

- 패산에서 1장을 쯔모한다
- 쯔모한 패가 대기패여서 쯔모 화료!
