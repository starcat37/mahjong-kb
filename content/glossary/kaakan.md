# 가깡 (加槓 / kaakan)

| 항목 | 내용 |
|------|------|
| 일본어 | 加槓 (カカン) |
| 로마자 | kaakan |
| 한국어 | 가깡 |

## 정의
가깡(加槓/소명깡)은 이미 퐁한 밍커에 같은 패 1장을 추가하여 깡으로 만드는 것.

## 상세 설명
퐁으로 만든 커쯔에 4번째 패를 추가하여 깡쯔로 승격. 가깡 선언 시 다른 플레이어가 창깡(chankan)으로 론할 수 있어 위험할 수 있다. 부수는 밍깡과 동일하게 취급.

## 사용 예시
- "퐁한 발에 4번째 발을 추가하여 가깡"
- "가깡 시 창깡에 주의해야 한다"
