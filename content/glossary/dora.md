# 도라 (ドラ / dora)

| 항목 | 내용 |
|------|------|
| 일본어 | ドラ |
| 로마자 | dora |
| 한국어 | 도라 |

## 정의

도라(ドラ)는 보유하고 있으면 1장당 1판이 추가되는 보너스 패. 역은 아니지만 점수를 올려준다.

## 상세 설명

왕패의 도라 표시패 다음 패가 도라. 예를 들어 표시패가 3만이면 4만이 도라. 자패는 동→남→서→북→동, 백→발→중→백 순으로 순환. 도라 자체는 역이 아니므로 도라만으로는 화료 불가.

## 사용 예시

- 도라 표시패가 5통이면 6통이 도라
- 도라 3장이면 3판 추가
