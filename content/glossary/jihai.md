# 자패 (字牌 / jihai)

| 항목 | 내용 |
|------|------|
| 일본어 | 字牌 (ジハイ) |
| 로마자 | jihai |
| 한국어 | 자패 |

## 정의

자패는 풍패(風牌) 4종과 삼원패(三元牌) 3종으로 구성된 총 7종 28장의 패이다.

## 상세 설명

풍패는 동(東), 남(南), 서(西), 북(北)이고, 삼원패는 백(白), 발(發), 중(中)이다. 자패는 슌쯔를 만들 수 없고, 커쯔나 또이쯔로만 사용할 수 있다. 모든 자패는 요구패(yaochuuhai)에 해당한다.

## 사용 예시

- "자패로는 슌쯔를 만들 수 없다."
- "쯔이소(자일색)는 자패만으로 화료하는 역만이다."
