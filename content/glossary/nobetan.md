# 노베탄 (延べ単 / nobetan)

| 항목 | 내용 |
|------|------|
| 일본어 | 延べ単 |
| 로마자 | nobetan |
| 한국어 | 노베탄 |

## 정의

양쪽으로 머리 단기를 기다리는 형태. 단기(単騎)가 뻗치다(延べる)라는 의미의 약어. e.g.) 1234의 1, 4 대기.
