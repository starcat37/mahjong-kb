# 적도라 (赤ドラ / akadora)

| 항목 | 내용 |
|------|------|
| 일본어 | 赤ドラ (あかドラ) |
| 로마자 | akadora |
| 한국어 | 적도라 |

## 정의

적도라(赤ドラ)는 빨간색으로 칠해진 5만·5통·5삭 패로, 보유 시 1장당 1판이 추가되는 보너스 패.

## 상세 설명

일반적으로 각 수패의 5에서 1장씩 교체하여 총 3장(5만·5통·5삭 각 1장)을 사용. 적도라는 일반 도라와 마찬가지로 역이 아닌 보너스 판수. 로컬 룰에 따라 개수가 다를 수 있다.

## 사용 예시

- 적5통을 들고 있으면 1판 추가
- 적도라는 판수를 올리지만 역 대용은 안 된다
