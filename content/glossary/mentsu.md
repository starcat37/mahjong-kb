# 멘쯔 (面子 / mentsu)

| 항목 | 내용 |
|------|------|
| 일본어 | 面子 (メンツ) |
| 로마자 | mentsu |
| 한국어 | 멘쯔 |

## 정의

멘쯔(面子)는 3장(또는 4장) 한 조를 이루는 패의 조합으로, 슌쯔·커쯔·깡쯔의 총칭이다.

## 상세 설명

일반적인 화료 형태는 멘쯔 4개 + 머리(또이쯔) 1개이다. 다른 플레이어에게서 울어서(나키) 만든 멘쯔는 밍커/밍깡이 되며 멘젠 상태가 깨진다. 치또이쯔(칠대자)처럼 멘쯔 없이 또이쯔 7개로도 화료가 가능하다.

## 사용 예시

- "멘쯔 4개와 머리 1개가 기본 화료 형태이다."
- "멘쯔가 부족하면 대기가 나빠진다."
