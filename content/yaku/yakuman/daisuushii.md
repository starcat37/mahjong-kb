# 다이스시 (大四喜)

| 항목 | 내용 |
|------|------|
| 일본어 | 大四喜 |
| 로마자 | daisuushii |
| 한국어 | 다이스시 |
| 판수 | 역만 (더블역만 가능) |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

다이스시는 풍패(동, 남, 서, 북) 4종 모두를 몸통(커쯔 또는 깡쯔)으로 갖추는 역만이다. 쇼스시의 상위 호환이며, 많은 룰에서 더블역만으로 인정된다. 책임지불(파오) 규칙이 적용된다.

## 성립 조건

1. 풍패 4종(東, 南, 西, 北) 모두를 커쯔 또는 깡쯔로 갖춰야 한다.
2. 머리(작두) 1조를 자유롭게 구성한다.
3. 멘젠, 후로 모두 가능하며 판수는 동일하다.

## 책임지불 (파오)

- 이미 풍패 3종을 퐁(또는 깡)으로 확보한 플레이어에게 4번째 풍패를 버려 퐁시킨 자가 **책임지불** 대상이 된다.
- 쯔모 화료 시: 책임지불자가 전액 부담.
- 론 화료 시: 방총자와 책임지불자가 절반씩 부담.

## 예시

### 기본 예시

```
東東東 南南南 西西西 北北北 5m5m
```
동동동 + 남남남 + 서서서 + 북북북 + 만수 55 머리

```
東東東 南南南 西西西 北北北 白白
```
동동동 + 남남남 + 서서서 + 북북북 + 백백 머리

### 주의할 패턴

- 풍패 4종 중 하나가 머리이면 다이스시가 아닌 쇼스시가 된다.
- 더블역만으로 인정하는 룰이 일반적이나, 싱글 역만만 인정하는 룰도 있다.
- 풍패 3종을 노출한 상대에게 4번째 풍패를 버리면 책임지불 대상이 되므로 극도로 주의한다.

## 전략적 활용

- 배패에 풍패가 대량으로 몰려 있을 때만 현실적으로 노릴 수 있다.
- 쇼스시를 진행하다가 머리로 계획한 풍패가 커쯔가 되면 다이스시로 발전한다.
- 후로를 적극 활용해 속도를 올리되, 상대의 경계를 감안해야 한다.
- 풍패 3종이 노출되면 다른 플레이어들이 4번째 풍패를 철저히 지키므로, 쯔모에 의존하는 경우가 많다.
