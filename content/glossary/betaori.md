# 베타오리 (ベタ降り / betaori)

| 항목 | 내용 |
|------|------|
| 일본어 | ベタ降り |
| 로마자 | betaori |
| 한국어 | 베타오리 |

## 정의

자신의 화료를 완전히 포기하고 손패를 무너뜨려가며 수비하는 행위. 한국에서는 '완전히 내린다'는 표현도 종종 쓰인다.
