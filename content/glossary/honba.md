# 본장 (本場 / honba)

| 항목 | 내용 |
|------|------|
| 일본어 | 本場 (ほんば) |
| 로마자 | honba |
| 한국어 | 본장 |

## 정의

본장(本場)은 연장이나 유국으로 누적되는 카운터로, 화료 시 추가 점수를 결정한다.

## 상세 설명

1본장당 론 화료는 300점, 쯔모 화료는 각 플레이어에게 100점씩 추가. 본장은 친 이외의 플레이어가 화료하면 0으로 리셋된다. 본장이 높을수록 낮은 역이라도 의미 있는 점수가 되어 빠른 화료의 가치가 올라간다.

## 사용 예시

- 3본장에서 론하면 +900점 보너스
- 본장이 쌓이면 빠른 화료가 더 가치 있다
