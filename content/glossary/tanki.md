# 단기대기 (単騎待ち / tanki)

| 항목 | 내용 |
|------|------|
| 일본어 | 単騎待ち (タンキまち) |
| 로마자 | tanki |
| 한국어 | 단기대기 |

## 정의
단기대기(単騎待ち)는 머리(또이쯔)가 될 1장을 기다리는 대기 형태.

## 상세 설명
4멘쯔가 완성된 상태에서 머리 1장을 기다린다. 대기패 1종 최대 3장(1장은 자신이 보유). 부수 계산에서 대기 부수 2부. 자패 단기는 읽기 어려워 전략적으로 활용할 수 있다.

## 사용 예시
- "멘쯔 4개 완성, 중 단기대기"
- "자패 단기는 상대가 읽기 어렵다"
