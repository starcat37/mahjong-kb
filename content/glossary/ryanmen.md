# 양면대기 (両面待ち / ryanmen)

| 항목 | 내용 |
|------|------|
| 일본어 | 両面待ち (リャンメンまち) |
| 로마자 | ryanmen |
| 한국어 | 양면대기 |

## 정의
양면대기(両面待ち)는 연속된 2장의 수패 양쪽 끝 패를 기다리는 대기 형태.

## 상세 설명
예를 들어 4-5를 들고 있으면 3과 6을 대기. 가장 효율이 좋은 대기로, 대기패가 2종 최대 8장. 핑후(pinfu)는 양면대기여야 성립한다. 부수 계산에서 대기 부수 0부.

## 사용 예시
- "4만-5만으로 3만·6만 양면대기"
- "양면대기는 화료 확률이 가장 높다"
