# 변짱대기 (辺張待ち / penchan)

| 항목 | 내용 |
|------|------|
| 일본어 | 辺張待ち (ペンチャンまち) |
| 로마자 | penchan |
| 한국어 | 변짱대기 |

## 정의
변짱대기(辺張待ち)는 수패 끝자리의 1-2로 3을, 또는 8-9로 7을 기다리는 대기 형태.

## 상세 설명
한쪽으로만 대기가 열려 있어 대기패 1종 최대 4장. 간짱과 마찬가지로 대기 부수 2부. 패효율이 낮아 가능하면 양면으로 전환하는 것이 좋다.

## 사용 예시
- "1만-2만으로 3만 변짱대기"
- "변짱은 빨리 해소하고 양면으로 바꾸는 게 유리"
