# 아가리야메 (和了り止め / agari yame)

| 항목 | 내용 |
|------|------|
| 일본어 | 和了り止め |
| 로마자 | agari yame |
| 한국어 | 아가리야메 |

## 정의

오라스에 친이 1등인 상태에서 화료하는 경우 연장 없이 경기를 종료할 권한을 가지는 룰. 넷마작에서는 아가리야메가 적용되더라도 연장 여부 선택 없이 즉시 종료되는 게 일반적.
