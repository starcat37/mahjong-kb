# 텐파이야메 (聴牌止め / tenpai yame)

| 항목 | 내용 |
|------|------|
| 일본어 | 聴牌止め |
| 로마자 | tenpai yame |
| 한국어 | 텐파이야메 |

## 정의

오라스 때 유국이 될 경우 1위인 오야가 텐파이인 상태라면 연장하지 않고 경기를 종료할 수 있는 권한을 가지는 룰.
