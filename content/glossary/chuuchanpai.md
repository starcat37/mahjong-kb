# 중장패 (中張牌 / chuuchanpai)

| 항목 | 내용 |
|------|------|
| 일본어 | 中張牌 (チュウチャンパイ) |
| 로마자 | chuuchanpai |
| 한국어 | 중장패 |

## 정의

중장패는 수패의 2~8에 해당하는 패를 통칭하는 용어이다.

## 상세 설명

1·9(노두패)와 자패(요구패)를 제외한 나머지 패를 가리킨다. 탕야오(tanyao)는 중장패로만 화료해야 성립하는 역이다. 중장패는 슌쯔를 만들기 쉬워 패효율이 높다.

## 사용 예시

- "탕야오를 노리려면 중장패 위주로 모아야 한다."
- "중장패는 대기가 양면이 되기 쉽다."
