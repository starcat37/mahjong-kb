# 국사무쌍 (国士無双)

| 항목 | 내용 |
|------|------|
| 일본어 | 国士無双 |
| 로마자 | kokushi musou |
| 한국어 | 국사무쌍 |
| 판수 | 역만 |
| 후로 가능 | 불가 (멘젠만) |
| 쿠이사가리 | 해당 없음 |

## 개요

국사무쌍은 요구패(노두패 + 자패) 13종을 각 1장씩 모으고, 그중 1종을 중복시켜 머리로 삼는 특수 형태의 역만이다. 일반적인 4몸통 + 1머리 구조와 완전히 다른 독립적인 화료 형태를 가진다. 13면대기 시 더블역만으로 인정하는 룰이 있다.

## 성립 조건

1. 멘젠(문전) 상태여야 한다.
2. 요구패 13종(1m, 9m, 1p, 9p, 1s, 9s, 東, 南, 西, 北, 白, 發, 中)을 각 1장씩 보유해야 한다.
3. 13종 중 1종을 추가로 1장 더 가져 머리(작두)로 삼는다.
4. 총 14장 구성: 요구패 13종 × 1장 + 머리 1장.

## 예시

### 기본 예시

```
1m 9m 1p 9p 1s 9s 東 南 西 北 白 發 中 + 1m
```
요구패 13종 + 만수 1이 머리

```
1m 9m 1p 9p 1s 9s 東 南 西 北 白 發 中 + 中
```
요구패 13종 + 중이 머리

### 주의할 패턴

- 13종이 모두 1장씩인 상태(13면대기)에서 화료하면 더블역만으로 인정되는 룰이 있다.
- 상대의 안깡에 대해서도 론이 가능한 특수 규칙이 적용된다(국사무쌍 안깡 론).
- 요구패 13종 중 하나라도 빠지면 성립하지 않는다.

## 전략적 활용

- 배패에 요구패가 9~10종 이상 있으면 적극적으로 노릴 수 있다.
- 중반까지 요구패 수집이 순조롭지 않으면 과감히 포기하고 수비로 전환하는 것이 좋다.
- 13면대기가 완성되면 어떤 요구패든 화료할 수 있어 대기 범위가 매우 넓다.
- 상대의 리치에 대해 안전패를 확보하기 어려우므로 속도 싸움이 중요하다.
