# 탕야오 (断幺九)

| 항목 | 내용 |
|------|------|
| 일본어 | 断幺九 |
| 로마자 | tanyao |
| 한국어 | 탕야오 |
| 판수 | 1판 |
| 후로 가능 | 가능 (쿠이탕 룰 적용 시) |
| 쿠이사가리 | 없음 |

## 개요

탕야오는 손패 전체를 중장패(2~8)만으로 구성하는 역이다. 1, 9, 자패를 하나도 포함하지 않아야 하며, 현대 리치마작에서는 울어도 인정하는 쿠이탕 룰이 주류이다.

## 성립 조건

1. 손패의 모든 패가 중장패(2~8)여야 한다.
2. 머리, 멘쯔 모두 1, 9, 자패가 포함되면 안 된다.
3. 쿠이탕 룰이 적용되는 경우 울어도(후로) 성립한다.

## 예시

### 기본 예시

만수 234, 통수 567, 삭수 345, 삭수 678, 통수 22 — 모든 패가 2~8 사이의 수패로만 구성되어 탕야오 성립.

### 주의할 패턴

- 만수 123처럼 1이 포함된 슌쯔가 있으면 탕야오가 불성립한다.
- 자패(동남서북백발중)가 머리라도 포함되면 불성립이다.
- 쿠이탕 룰을 적용하지 않는 탁에서는 멘젠 상태에서만 성립한다.

## 전략적 활용

- 가장 만들기 쉬운 역 중 하나로, 속공에 적합하다.
- 폰이나 치로 빠르게 완성할 수 있어 선제 텐파이를 노릴 때 유용하다.
- 핑후, 이페코 등 멘젠 역과 복합하면 판수를 쉽게 올릴 수 있다.
- 초보자는 배패에서 중장패가 많을 때 탕야오를 목표로 잡는 연습을 추천한다.
