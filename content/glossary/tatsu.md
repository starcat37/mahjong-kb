# 타쯔 (塔子 / tatsu)

| 항목 | 내용 |
|------|------|
| 일본어 | 塔子 |
| 로마자 | tatsu |
| 한국어 | 타쯔 |

## 정의

특정 패가 한 장만 더 있으면 슌쯔가 되는, 연속되거나 한 장 떨어진 수패 2장을 부르는 말. e.g.) 12(펜타쯔), 35(칸타쯔), 67(량타쯔).
