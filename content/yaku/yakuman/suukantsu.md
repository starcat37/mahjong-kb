# 스깡쯔 (四槓子)

| 항목 | 내용 |
|------|------|
| 일본어 | 四槓子 |
| 로마자 | suukantsu |
| 한국어 | 스깡쯔 |

| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

스깡쯔는 깡을 4회 선언하여 깡쯔 4개와 머리 1조를 갖추는 역만이다. 리치마작에서 가장 달성 난이도가 높은 역 중 하나이며, 안깡과 밍깡(다이밍깡, 쇼밍깡)을 조합하여 구성한다. 책임지불(파오) 규칙이 적용된다.

## 성립 조건

1. 깡쯔 4개를 갖춰야 한다(안깡, 밍깡 불문).
2. 머리(작두) 1조가 필요하다.
3. 4번째 깡 선언 후 린샨패로 화료하거나, 이후 텐파이 상태에서 화료해야 한다.
4. 타가(다른 플레이어)가 1회라도 깡을 선언하면 스깡쯔가 불가능해진다(사깡산료로 유국).

## 책임지불 (파오)

- 이미 깡을 3회 선언한 플레이어에게 4번째 깡의 대상패를 버린 자가 **책임지불** 대상이 된다.
- 쯔모 화료 시: 책임지불자가 전액 부담.
- 론 화료 시: 방총자와 책임지불자가 절반씩 부담.

## 예시

### 기본 예시

```
[1m1m1m1m] [5p5p5p5p] [9s9s9s9s] [東東東東] 白白
```
만수 1111(깡) + 통수 5555(깡) + 삭수 9999(깡) + 동동동동(깡) + 백백 머리

```
[3m3m3m3m] [7m7m7m7m] [2p2p2p2p] [中中中中] 6s6s
```
만수 3333(깡) + 만수 7777(깡) + 통수 2222(깡) + 중중중중(깡) + 삭수 66 머리

### 주의할 패턴

- 4번의 깡 모두 한 사람이 해야 한다. 타가가 깡을 1회라도 하면 4회째 깡 시점에서 사깡산료(유국)가 된다.
- 깡 4회에 필요한 린샨패(영상패) 소비와 왕패 보충에 주의해야 한다.
- 실전에서 달성되는 경우는 극히 드물다.

## 전략적 활용

- 의도적으로 노리기보다는 깡이 3회 이상 자연스럽게 발생했을 때 추가로 노리는 형태가 현실적이다.
- 3번째 깡이 성공하면 상대 플레이어들은 4번째 깡의 대상패를 절대 버리지 않으려 한다.
- 안깡 중심으로 진행하면 상대에게 읽히기 어렵다.
