# 마타기스지 (跨ぎ筋 / matagi suji)

| 항목 | 내용 |
|------|------|
| 일본어 | 跨ぎ筋 |
| 로마자 | matagi suji |
| 한국어 | 마타기스지 |

## 정의

버림패를 포함한 양면대기의 스지패. 양면 변화를 가장했을 때의 상대의 리치선언패, 버림패 기준의 위험패. e.g.) 5를 버린 상대에게 (556에서 5, 455에서 5를 타패했을 가능성이 있으므로) 36, 47은 위험패.
