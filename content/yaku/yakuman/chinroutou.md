# 친로토 (清老頭)

| 항목 | 내용 |
|------|------|
| 일본어 | 清老頭 |
| 로마자 | chinroutou |
| 한국어 | 친로토 |
| 판수 | 역만 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

친로토는 수패의 1과 9(노두패)만으로 손패 전체를 구성하는 역만이다. 자패를 사용하지 않으며, 슌쯔를 만들 수 없으므로 또이또이(대대화) 형태가 된다. 후로해도 역만이 유지된다.

## 성립 조건

1. 손패의 모든 몸통과 머리가 수패 1 또는 9로만 이루어져야 한다.
2. 자패는 사용할 수 없다.
3. 슌쯔는 불가능하므로, 커쯔(또는 깡쯔) + 머리 형태만 가능하다.
4. 사용 가능한 패: 1m, 9m, 1p, 9p, 1s, 9s (총 6종).

## 예시

### 기본 예시

```
1m1m1m 9m9m9m 1p1p1p 9s9s9s 1s1s
```
만수 111 + 만수 999 + 통수 111 + 삭수 999 + 삭수 11 머리

```
1m1m1m 9p9p9p 1s1s1s 9s9s9s 9m9m
```
만수 111 + 통수 999 + 삭수 111 + 삭수 999 + 만수 99 머리

### 주의할 패턴

- 자패가 하나라도 포함되면 친로토가 아닌 혼로토(혼노두)가 된다.
- 사용 가능한 패가 6종뿐이므로 구성 난이도가 매우 높다.
- 또이또이, 준찬타 등은 자동으로 포함되지만 역만이므로 추가 판수를 계산하지 않는다.

## 전략적 활용

- 배패에 1, 9 수패가 대량으로 몰려 있을 때만 현실적으로 노릴 수 있다.
- 사용 패가 6종뿐이므로 다른 플레이어의 버림패에서 퐁할 기회가 상대적으로 많다.
- 혼로토나 준찬타를 진행하다가 자패를 버리고 친로토로 전환하는 경우가 있다.
- 상대가 1, 9패를 다수 퐁하면 친로토를 경계해야 한다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 친로토가 포함되어 있을 확률은 약 **0.0011%**이다.
