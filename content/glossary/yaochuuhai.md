# 요구패 (么九牌 / yaochuuhai)

| 항목 | 내용 |
|------|------|
| 일본어 | 么九牌 (ヤオチューハイ) |
| 로마자 | yaochuuhai |
| 한국어 | 요구패 |

## 정의

요구패(么九牌)는 수패의 1·9와 모든 자패를 총칭하는 용어이다.

## 상세 설명

요구패에 해당하는 수패의 1과 9를 노두패(老頭牌)라 하고, 풍패와 삼원패를 합쳐 자패라 한다. 혼노두(honroutou), 국사무쌍(kokushi) 등의 역에서 핵심 역할을 한다. 요구패가 아닌 2~8의 수패는 중장패(chuuchanpai)라 한다.

## 사용 예시

- "국사무쌍은 13종의 요구패를 모두 모아야 한다."
- "찬타는 모든 멘쯔에 요구패가 포함되어야 한다."
