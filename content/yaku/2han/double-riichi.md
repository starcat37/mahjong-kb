# 더블리치 (ダブル立直)

| 항목 | 내용 |
|------|------|
| 일본어 | 両立直 (ダブル立直) |
| 로마자 | daburu riichi |
| 한국어 | 더블리치 |
| 판수 | 2판 |
| 후로 가능 | 불가 |
| 쿠이사가리 | 없음 |

## 개요

더블리치는 자신의 첫 번째 쯔모 직후에 리치를 선언하는 역이다. 배패 시점에서 이미 텐파이 상태여야 하므로 운의 요소가 매우 크며, 통상 리치(1판)보다 1판 높은 2판이 부여된다.

## 성립 조건

1. 자신의 첫 번째 쯔모 차례에 리치를 선언해야 한다.
2. 자신의 첫 쯔모 전까지 다른 플레이어가 치, 퐁, 깡 등 후로를 하지 않은 상태여야 한다.
3. 멘젠(문전) 상태여야 한다.
4. 리치 선언 시 1000점 공탁금을 지불해야 한다.

## 예시

### 기본 예시

배패: 만수 234, 만수 567, 통수 33, 삭수 789, 삭수 46
첫 쯔모에서 삭수 5를 쯔모 → 텐파이 확인 후 리치 선언 → 더블리치 성립

### 주의할 패턴

- 자신의 첫 쯔모 전에 다른 플레이어가 치나 퐁을 했다면 더블리치가 아닌 통상 리치만 가능하다.
- 배패 시점에서 텐파이가 아니라면 첫 쯔모로 텐파이가 되더라도 더블리치 선언이 가능하다 (첫 쯔모 후 판단).

## 전략적 활용

- 더블리치는 의도적으로 노릴 수 있는 역이 아니므로, 배패가 좋을 때 놓치지 않는 것이 중요하다.
- 더블리치 + 잇파쯔가 성립하면 3판이 되어 높은 점수를 기대할 수 있다.
- 쯔모로 화료하면 더블리치(2판) + 멘젠쯔모(1판) + 잇파쯔(1판) = 4판까지 가능하다.
