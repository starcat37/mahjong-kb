# 통수패 (筒子 / pinzu)

| 항목 | 내용 |
|------|------|
| 일본어 | 筒子 (ピンズ) |
| 로마자 | pinzu |
| 한국어 | 통수패 |

## 정의

통수패는 리치마작에서 사용하는 세 가지 수패 중 하나로, 동그란 원(통) 무늬가 새겨져 있으며 1통부터 9통까지 각 4장씩 총 36장으로 구성된다.

## 상세 설명

통수패는 원형 무늬의 개수로 숫자를 나타낸다. 일본어로 핀즈(ピンズ)라고 불린다. [[manzu]](만수패), [[souzu]](삭수패)와 함께 수패(数牌)를 구성하며, 슌쯔나 커쯔를 만들 수 있다.

## 사용 예시

- "통수패 1, 2, 3으로 슌쯔를 완성했다."
- "핀즈의 5는 적도라로 쓰이는 경우가 많다."
