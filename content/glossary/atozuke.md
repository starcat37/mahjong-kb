# 아토즈케 (後付け / atozuke)

| 항목 | 내용 |
|------|------|
| 일본어 | 後付け |
| 로마자 | atozuke |
| 한국어 | 아토즈케 |

## 정의

손패 안에 역을 확정짓지 않은 상태에서 후로하는 것을 뜻한다. '나중에 역을 붙인다(後から役を付ける)'가 어원. 실전에서는 역패 또이쯔를 가지고 있는 상황에서 다른 블럭부터 후로하는 식이 가장 잦다.
