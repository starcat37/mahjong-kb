# 치또이쯔 (七対子)

| 항목 | 내용 |
|------|------|
| 일본어 | 七対子 |
| 로마자 | chiitoitsu |
| 한국어 | 치또이쯔 |
| 판수 | 2판 |
| 후로 가능 | 불가 |
| 쿠이사가리 | 없음 |

## 개요

치또이쯔는 7개의 또이쯔(대자, 같은 패 2장)로 손패를 구성하는 특수형 역이다. 일반적인 4멘쯔 + 1머리 형태가 아닌 독자적인 화료 형태로, 부수 계산은 25부 고정이며 멘젠 전용 역이다.

## 성립 조건

1. 7개의 서로 다른 종류의 또이쯔(같은 패 2장)로 손패를 구성해야 한다.
2. 같은 패 4장을 2개의 또이쯔로 사용하는 것은 인정되지 않는다.
3. 반드시 7종류의 서로 다른 패로 또이쯔를 만들어야 한다.
4. 후로할 수 없다 (멘젠 전용).
5. 부수 계산은 25부 고정이다.

## 예시

### 기본 예시

만수 11, 만수 55, 통수 33, 통수 99, 삭수 22, 동동, 발발
→ 7종류의 또이쯔로 구성되었으므로 치또이쯔 성립

### 주의할 패턴

- 만수 1111처럼 같은 패 4장이 있을 때, 이를 또이쯔 2개(만수 11 + 만수 11)로 세는 것은 불가능하다.
- 치또이쯔는 또이또이와 동시에 성립하지 않는다 (또이또이는 커쯔 4개 + 머리 1개 형태).

## 전략적 활용

- 배패에서 또이쯔가 많고 슌쯔 구성이 어려울 때 방향 전환할 수 있는 역이다.
- 25부 2판 고정이므로 다른 역과 복합하여 판수를 올리는 것이 중요하다.
- 탕야오, 호닛쯔 등과 복합이 가능하다.
- 텐파이 시 반드시 단기대기(1종류 대기)가 되므로 대기패 선택이 중요하다.
- 리치와 복합하면 3판 + 잇파쯔나 우라도라 기대가 가능하다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 치또이쯔가 포함되어 있을 확률은 약 **2.60%**이다. 국사무쌍과 함께 3멘쯔 + 1머리가 아닌 특수 화료 형태이다.
