# 판수 (翻 / han)

| 항목 | 내용 |
|------|------|
| 일본어 | 翻 (はん) |
| 로마자 | han |
| 한국어 | 판수 |

## 정의

판수(翻/はん)는 역과 도라로 결정되는 점수 배율로, 부수와 함께 최종 점수를 결정한다.

## 상세 설명

역마다 정해진 판수가 있으며(예: 리치 1판, 치또이쯔 2판), 도라는 1장당 1판 추가. 판수가 높을수록 점수가 기하급수적으로 증가하며, 5판 이상은 만관 이상의 고정 점수가 된다. 쿠이사가리(식하)가 있는 역은 울면 1판 내려간다.

## 사용 예시

- "리치 1판 + 도라 2판 = 3판"
- "5판 이상은 만관으로 고정 점수 적용"
