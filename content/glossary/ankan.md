# 안깡 (暗槓 / ankan)

| 항목 | 내용 |
|------|------|
| 일본어 | 暗槓 (アンカン) |
| 로마자 | ankan |
| 한국어 | 안깡 |

## 정의
안깡(暗槓)은 울기 없이 자력으로 모은 같은 패 4장으로 선언하는 깡.

## 상세 설명
멘젠 상태를 유지한 채 깡을 할 수 있는 유일한 방법. 단, 리치 후 안깡은 대기가 변하지 않는 경우에만 가능. 부수 계산에서 안깡은 가장 높은 부수(중장패 16부, 요구패 32부)를 받는다.

## 사용 예시
- "북 4장을 모아 안깡 선언"
- "안깡은 멘젠이 유지되어 리치와 병행 가능"
