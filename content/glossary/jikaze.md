# 자풍 (自風 / jikaze)

| 항목 | 내용 |
|------|------|
| 일본어 | 自風 |
| 로마자 | jikaze |
| 한국어 | 자풍 |

## 정의

그 국에서 자기 자신의 바람을 가리키는 말. 동가일 때는 東, 서가일 때는 西, 남가일 때는 南, 북가일 때는 北이 자풍이 된다. 자풍이 아닌 바람은 객풍(오타카제).
