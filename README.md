# 리치마작 종합 백과사전 (majhong-kb)

리치마작(일본식 마작) 종합 백과사전 Markdown 기반 지식 베이스.

## 특징

- Obsidian 볼트로 바로 열람 가능
- 체계적인 frontmatter 스키마로 구조화
- Python RAG 파이프라인 연동 예정

## 구조

```
content/
├── rules/        # 규칙 기초 (패 종류, 게임 진행, 멘젠/후로, 화료 조건)
├── yaku/         # 역(야쿠) — 판수별 분류
├── scoring/      # 점수 계산 (부수, 판수, 만관)
├── glossary/     # 용어 사전
├── strategy/     # 전략/전술
├── pro/          # 프로/대회
├── culture/      # 문화/역사
└── faq/          # FAQ
```

## 사용법

1. Obsidian에서 `content/` 폴더를 볼트로 열기
2. `_templates/` 폴더를 Obsidian 템플릿 폴더로 지정
3. 새 문서 작성 시 해당 템플릿 적용
