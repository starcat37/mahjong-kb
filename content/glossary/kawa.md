# 강 (河 / kawa)

| 항목 | 내용 |
|------|------|
| 일본어 | 河 |
| 로마자 | kawa |
| 한국어 | 강 |

## 정의

버림패를 늘어놓는 곳. 일본어로는 '호(河)'라고 발음한다. '바닥이 강하다'는 표현은 스지가 거의 뚫려있지 않아 잠재적인 위험패를 손 안에 묶어두기 어려운 바닥을 뜻한다. 반대 상황에는 '바닥이 약하다' '바닥이 미지근하다(河がぬるい)'라는 표현을 쓴다.
