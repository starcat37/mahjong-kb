# 화료 (和了 / agari)

| 항목 | 내용 |
|------|------|
| 일본어 | 和了 (あがり) |
| 로마자 | agari |
| 한국어 | 화료 |

## 정의
화료(和了)는 패를 완성하여 승리하는 것. 론(栄)과 쯔모(ツモ) 두 가지 방법이 있다.

## 상세 설명
기본 화료 형태는 4멘쯔+1또이쯔. 화료하려면 최소 1개 이상의 역(야쿠)이 필요하다. 화료 시 점수를 계산하여 다른 플레이어로부터 점수를 받는다.

## 사용 예시
- "역이 없으면 텐파이여도 화료할 수 없다"
- "쯔모 화료는 전원에게서 점수를 받는다"
