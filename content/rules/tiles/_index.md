# 패 종류

리치마작에서 사용하는 패(牌, はい)는 총 136장(34종 × 4장)입니다.

## 수패 (数牌)
- **만수패** (萬子, ワンズ): 1만~9만
- **통수패** (筒子, ピンズ): 1통~9통
- **삭수패** (索子, ソーズ): 1삭~9삭

## 자패 (字牌)
- **풍패** (風牌): 동(東), 남(南), 서(西), 북(北)
- **삼원패** (三元牌): 백(白), 발(發), 중(中)
