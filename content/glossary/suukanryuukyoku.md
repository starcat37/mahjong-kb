# 사깡유국 (四槓流局 / suukan ryuukyoku)

| 항목 | 내용 |
|------|------|
| 일본어 | 四槓流局 |
| 로마자 | suukan ryuukyoku |
| 한국어 | 사깡유국 |

## 정의

국 진행 도중 2명 이상에 의해 깡이 4번 선언된 경우를 뜻하는 말. 유국 처리된다. 한 사람이 4번의 깡을 발성한 경우에는 유국되지 않고 그대로 진행된다.
