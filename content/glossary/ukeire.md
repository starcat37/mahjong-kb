# 우케이레 (受け入れ / ukeire)

| 항목 | 내용 |
|------|------|
| 일본어 | 受け入れ |
| 로마자 | ukeire |
| 한국어 | 우케이레 |

## 정의

가져왔을 때 샨텐 수가 떨어지는(=손이 진행되는, 텐파이에 가까워지는) 패. '유효패'라는 표현과 혼용되는 경우가 많다.
