# 깡도라 (槓ドラ / kandora)

| 항목 | 내용 |
|------|------|
| 일본어 | 槓ドラ (カンドラ) |
| 로마자 | kandora |
| 한국어 | 깡도라 |

## 정의

깡도라(槓ドラ)는 깡이 선언될 때마다 왕패에서 추가로 공개되는 도라 표시패로 결정되는 도라.

## 상세 설명

깡 1회마다 도라 표시패 1개가 추가되어 최대 4개까지 늘어날 수 있다. 리치 상태에서 깡도라가 추가되면 뒷도라도 함께 늘어난다. 깡은 도라가 늘어나는 양날의 검.

## 사용 예시

- 깡을 했더니 깡도라가 자신의 패에 3장!
- 상대의 깡으로 깡도라가 내게 유리하게 열릴 수도
