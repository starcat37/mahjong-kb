# 스지 카운트 (筋カウント / suji kaunto)

| 항목 | 내용 |
|------|------|
| 일본어 | 筋カウント |
| 로마자 | suji kaunto |
| 한국어 | 스지 카운트 |

## 정의

리치를 대상으로, 리치자의 버림패 및 이후 버려지는 수패를 통해 스지가 얼마나 통했는지 세어 무스지의 위험도를 가늠하는 기술. 수패 1종류에 6개의 스지가 있으므로 스지의 총 개수는 18개이며, 1/18 이론이라는 명칭은 여기서 유래.
