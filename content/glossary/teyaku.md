# 손역 (手役 / teyaku)

| 항목 | 내용 |
|------|------|
| 일본어 | 手役 |
| 로마자 | teyaku |
| 한국어 | 손역 |

## 정의

화료한 때, 혹은 지금 손패에 성립하는 역을 뜻하는 말. 특정한 역을 의식해 가며 조패하는 것을 테야쿠즈꾸리(手役作り)라고 한다.
