# 준찬타 (純全帯幺九)

| 항목 | 내용 |
|------|------|
| 일본어 | 純全帯幺九 |
| 로마자 | junchan |
| 한국어 | 준찬타 |
| 판수 | 멘젠 3판 / 후로 2판 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 있음 (후로 시 2판) |

## 개요

준찬타는 모든 몸통과 머리에 수패의 1 또는 9가 포함되며, 자패가 전혀 없는 역이다. 찬타에서 자패를 제외한 상위 호환 역으로, 멘젠 시 3판, 후로 시 2판이다.

## 성립 조건

1. 모든 몸통(멘쯔)과 머리에 수패 1 또는 9가 반드시 포함되어야 한다.
2. 수패 123 또는 789 형태의 슌쯔가 최소 1개 이상 포함되어야 한다.
3. 자패(풍패, 삼원패)는 사용할 수 없다.
4. 후로 시 쿠이사가리가 적용되어 2판으로 감소한다.

## 예시

### 기본 예시

```
1m2m3m 1p2p3p 7s8s9s 9m9m9m 1s1s
```
만수 123 + 통수 123 + 삭수 789 + 만수 999 + 삭수 11 머리

```
1m1m1m 7p8p9p 1s2s3s 7m8m9m 9p9p
```
만수 111 + 통수 789 + 삭수 123 + 만수 789 + 통수 99 머리

### 주의할 패턴

- 자패가 하나라도 포함되면 준찬타가 아닌 찬타가 된다.
- 모든 몸통이 커쯔(또는 깡쯔)이고 1, 9로만 구성되면 친로토(역만)가 된다.

## 전략적 활용

- 배패에 1, 9 수패가 많을 때 노려볼 수 있다.
- 후로해도 2판이 확보되므로, 속도를 중시할 때 적극적으로 울 수 있다.
- 핑후, 이페코 등 슌쯔 계열 역과 복합이 가능하다.
