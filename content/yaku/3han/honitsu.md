# 호닛쯔 (混一色)

| 항목 | 내용 |
|------|------|
| 일본어 | 混一色 |
| 로마자 | honitsu |
| 한국어 | 호닛쯔 |
| 판수 | 멘젠 3판 / 후로 2판 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 있음 (후로 시 2판) |

## 개요

호닛쯔는 한 종류의 수패와 자패만으로 손패를 구성하는 역이다. 멘젠 시 3판, 후로 시 2판으로, 염색 계열의 기본이 되는 역이며 실전에서 자주 등장한다.

## 성립 조건

1. 수패는 만수, 통수, 삭수 중 한 종류만 사용해야 한다.
2. 자패(풍패, 삼원패)를 함께 사용할 수 있다.
3. 후로 시 쿠이사가리가 적용되어 2판으로 감소한다.

## 예시

### 기본 예시

```
1m2m3m 5m6m7m 9m9m9m 東東東 白白
```
만수 123 + 만수 567 + 만수 999 + 동동동 + 백백 머리 (만수 + 자패)

```
2p3p4p 6p7p8p 中中中 南南南 1p1p
```
통수 234 + 통수 678 + 중중중 + 남남남 + 통수 11 머리 (통수 + 자패)

### 주의할 패턴

- 자패가 전혀 없이 수패 한 종류만으로 구성하면 친잇쯔(청일색)가 되어 6판이 된다.
- 수패가 2종류 이상 섞이면 호닛쯔가 성립하지 않는다.

## 전략적 활용

- 배패에 한 종류 수패가 집중되어 있고 자패도 있을 때 유력한 선택지이다.
- 야쿠하이(역패)와 복합이 쉬워 후로해도 높은 점수를 기대할 수 있다.
- 후로를 적극적으로 하면 상대에게 호닛쯔가 읽히기 쉬우므로 방어에 주의해야 한다.
- 손패가 진행되면서 자패를 모두 버리면 친잇쯔로의 전환도 가능하다.
