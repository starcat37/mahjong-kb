# 소바텐 (ソバテン / sobaten)

| 항목 | 내용 |
|------|------|
| 일본어 | ソバテン |
| 로마자 | sobaten |
| 한국어 | 소바텐 |

## 정의

마지막으로 버려진 테다시 패의 주변패를 대기로 잡은 텐파이를 뜻하는 말. e.g.) 2만을 버리고 23만으로 1-4만 대기.

## 관련 용어

- [[tedashi]] — 테다시
- [[tenpai]] — 텐파이
