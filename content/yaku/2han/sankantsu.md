# 산깡쯔 (三槓子)

| 항목 | 내용 |
|------|------|
| 일본어 | 三槓子 |
| 로마자 | sankantsu |
| 한국어 | 산깡쯔 |
| 판수 | 2판 |
| 후로 가능 | 가능 |
| 쿠이사가리 | 없음 |

## 개요

산깡쯔는 깡(같은 패 4장을 선언)을 3회 성공시켜 깡쯔 3개를 보유하는 역이다. 안깡, 밍깡, 가깡 어떤 종류든 상관없이 깡쯔 3개면 성립하며, 실전에서 출현 빈도가 극히 낮은 희귀한 역이다.

## 성립 조건

1. 깡쯔(같은 패 4장)를 3개 보유해야 한다.
2. 안깡, 밍깡(대명깡), 가깡(소명깡) 모두 인정된다.
3. 나머지 1개의 몸통은 커쯔, 슌쯔 등 자유롭게 구성할 수 있다.
4. 1개의 머리(또이쯔)가 필요하다.
5. 후로 가능하며 쿠이사가리가 적용되지 않는다.

## 예시

### 기본 예시

깡쯔: 만수 1111(안깡), 통수 5555(밍깡), 삭수 9999(가깡)
커쯔: 동동동
머리: 백백
→ 깡쯔 3개가 있으므로 산깡쯔 성립

### 주의할 패턴

- 한 국에서 깡은 최대 4회까지 가능하지만, 서로 다른 플레이어가 합계 4회째 깡을 하면 유국(사깡산료)이 된다. 단, 같은 플레이어가 4회 모두 하면 스깡쯔(역만)가 된다.
- 다른 플레이어가 2회 이상 깡을 하면 자신이 3회 깡을 할 수 없게 될 수 있다.
- 깡을 많이 하면 도라 표시패가 늘어나 다른 플레이어에게도 이득이 될 수 있어 위험하다.

## 전략적 활용

- 의도적으로 노리기 매우 어려운 역으로, 자연스럽게 깡이 3번 가능한 상황에서만 고려한다.
- 깡을 할 때마다 도라 표시패가 추가되므로, 도라가 자신의 손패에 유리하게 작용할 수 있다.
- 또이또이와 복합하기 쉽다 (깡쯔 3개 + 커쯔 1개).
- 린샨카이호(린샨패로 쯔모)와 복합될 가능성이 있다.
