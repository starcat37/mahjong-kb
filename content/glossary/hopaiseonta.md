# 호패선타 (好牌先打 / kouhai senta)

| 항목 | 내용 |
|------|------|
| 일본어 | 好牌先打 |
| 로마자 | kouhai senta |
| 한국어 | 호패선타 |

## 정의

내가 사용하지 않을 거라면, 다른 사람이 사용하기 쉬운(좋은) 패를 먼저 버린다는 원칙. 역패의 처리 순서나 타쯔를 버리는 순서에 영향을 준다.
