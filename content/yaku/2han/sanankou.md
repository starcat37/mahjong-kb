# 산안코 (三暗刻)

| 항목 | 내용 |
|------|------|
| 일본어 | 三暗刻 |
| 로마자 | san ankou |
| 한국어 | 산안코 |
| 판수 | 2판 |
| 후로 가능 | 가능 (안커 3개 필수, 나머지는 울어도 됨) |
| 쿠이사가리 | 없음 |

## 개요

산안코는 안커(손패 안에서 자력으로 모은 커쯔) 3개를 포함하는 역이다. 나머지 1개의 몸통은 슌쯔나 밍커(울어서 만든 커쯔)여도 상관없으며, 후로해도 판수 감소 없이 2판이 유지된다.

## 성립 조건

1. 안커(손패에서 자력으로 완성한 커쯔)가 3개 있어야 한다.
2. 나머지 1개의 몸통은 슌쯔, 밍커, 깡쯔 등 자유롭게 구성할 수 있다.
3. 1개의 머리(또이쯔)가 필요하다.
4. 쯔모로 화료하면 마지막 커쯔도 안커로 인정된다.
5. 론으로 화료하면 마지막 커쯔가 밍커 판정이 되므로 안커 수에 주의해야 한다.

## 예시

### 기본 예시

안커: 만수 222, 통수 555, 삭수 888 (이상 손패에서 완성)
밍커: 동동동 (퐁으로 획득)
머리: 발발
→ 안커 3개가 있으므로 산안코 성립

### 주의할 패턴

- 안커 2개 + 커쯔 텐파이 상태에서 론으로 화료하면, 마지막 커쯔는 밍커가 되어 안커가 2개뿐이므로 산안코가 불성립한다.
- 같은 형태라도 쯔모로 화료하면 안커 3개가 되어 산안코가 성립한다. 화료 방법에 따라 역의 성립 여부가 달라지는 대표적인 경우다.

## 전략적 활용

- 또이또이와 복합하기 쉽다. 안커 3개 + 밍커 1개라면 또이또이 + 산안코 = 4판이 된다.
- 멘젠에서 안커 3개 + 안커 텐파이 시 쯔모하면 스안커(역만)가 되므로, 쯔모를 노리는 것이 유리하다.
- 론 화료 시 안커 판정에 주의하여 텐파이 형태를 잘 관리해야 한다.

## 출현 확률

작혼 4인 반장전 통계 기준, 화료 시 산안코가 포함되어 있을 확률은 약 **0.67%**이다.
