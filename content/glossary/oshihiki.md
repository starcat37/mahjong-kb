# 오시히키 (押し引き / oshi hiki)

| 항목 | 내용 |
|------|------|
| 일본어 | 押し引き |
| 로마자 | oshi hiki |
| 한국어 | 오시히키 |

## 정의

마작에서 승부에 나설지 말지를 결정하는 기술. 한국에서는 보통 '공수판단'으로 의역한다.

## 관련 용어

- [[oshi]] — 오시
- [[ori]] — 오리
