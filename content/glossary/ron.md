# 론 (栄 / ron)

| 항목 | 내용 |
|------|------|
| 일본어 | 栄 (ロン) |
| 로마자 | ron |
| 한국어 | 론 |

## 정의
론(栄)은 다른 플레이어가 버린 패로 화료하는 것.

## 상세 설명
텐파이 상태에서 대기패를 다른 플레이어가 버리면 론 선언 가능. 방총자(론 당한 사람) 한 명이 점수를 모두 지불. 후리텐 상태에서는 론 불가.

## 사용 예시
- "상가가 버린 3만으로 론!"
- "후리텐이면 론을 할 수 없고 쯔모만 가능"
