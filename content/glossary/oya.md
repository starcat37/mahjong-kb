# 친 (親 / oya)

| 항목 | 내용 |
|------|------|
| 일본어 | 親 (おや) |
| 로마자 | oya |
| 한국어 | 친 |

## 정의

친(親)은 해당 국의 딜러로, 동(東)의 자리에 앉은 플레이어.

## 상세 설명

친은 배패 시 14장을 받고 먼저 타패한다. 친이 화료하면 점수가 자(子)보다 1.5배 높다. 친이 화료 또는 텐파이로 유국하면 연장(連荘)이 된다. 친이 화료하지 못하면 다음 플레이어에게 친이 넘어간다.

## 사용 예시

- 친의 만관은 12000점(자의 만관은 8000점)
- 친이 화료하면 연장으로 본장이 쌓인다
