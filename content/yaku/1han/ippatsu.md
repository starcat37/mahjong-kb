# 잇파쯔 (一発)

| 항목 | 내용 |
|------|------|
| 일본어 | 一発 |
| 로마자 | ippatsu |
| 한국어 | 잇파쯔 |
| 판수 | 1판 |
| 후로 가능 | 불가 |
| 쿠이사가리 | 없음 |

## 개요

잇파쯔는 리치를 선언한 후, 누구도 울지(치, 폰, 깡) 않은 상태에서 1순 이내에 화료하는 역이다. 리치에 부수적으로 따라오는 우연역으로, 리치 선언 후 빠르게 화료할 경우 자동으로 1판이 추가된다.

## 성립 조건

1. 리치를 선언한 상태여야 한다.
2. 리치 선언 후 자신의 다음 쯔모 차례까지(1순 이내) 화료해야 한다.
3. 그 사이에 누구도 치, 폰, 깡을 선언하지 않아야 한다.

## 예시

### 기본 예시

리치를 선언하고 다음 자신의 쯔모 차례에 대기패를 쯔모 — 리치 + 잇파쯔 + 멘젠쯔모로 최소 3판.

리치 선언 후, 다음 자신의 차례가 오기 전에 하가(아래쪽 플레이어)가 버린 패가 대기패와 일치하여 론 — 리치 + 잇파쯔로 2판.

### 주의할 패턴

- 리치 선언 후 누군가가 치, 폰, 깡을 하면 잇파쯔 기회가 소멸한다.
- 자신의 2번째 쯔모부터는 이미 1순이 지났으므로 잇파쯔가 아니다.
- 리치 선언 시 버린 패로 다른 사람이 론하는 것은 잇파쯔와 무관하다.

## 전략적 활용

- 잇파쯔를 의도적으로 노리기는 어렵지만, 리치 선언 자체에 잇파쯔 가능성이 내포되어 있어 리치의 기대값을 높여준다.
- 상대의 리치에 대해 잇파쯔를 방지하려면, 불필요하더라도 치나 폰을 선언하는 전략이 있다 (잇파쯔 소거).
- 초보자는 "리치 후 1순 안에 화료하면 1판 보너스"라고 기억하면 된다.
