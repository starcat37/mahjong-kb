# 형식 텐파이 (形式聴牌 / keishiki tenpai)

| 항목 | 내용 |
|------|------|
| 일본어 | 形式聴牌 |
| 로마자 | keishiki tenpai |
| 한국어 | 형식 텐파이 |

## 정의

손패 안에 역이 없어서 화료할 수는 없지만, 손패의 모양 자체는 텐파이인 상태. 주로 유국시 노텐 벌금을 지불하지 않기 위해 취한다.
