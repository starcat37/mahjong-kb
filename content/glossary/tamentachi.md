# 다면대기 (多面待ち / tamen machi)

| 항목 | 내용 |
|------|------|
| 일본어 | 多面待ち |
| 로마자 | tamen machi |
| 한국어 | 다면대기 |

## 정의

여러 대기가 합쳐져 3종 이상의 패를 기다리는 상태. e.g.) 23456의 147 대기, 4445의 365 대기.
