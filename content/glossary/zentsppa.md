# 젠츳파 (全ツッパ / zentsppa)

| 항목 | 내용 |
|------|------|
| 일본어 | 全ツッパ |
| 로마자 | zentsppa |
| 한국어 | 젠츳파 |

## 정의

방총의 위험을 무시하고, 자신의 화료만을 보면서 손을 만드는 행위. '전부 공격한다'는 뜻.
