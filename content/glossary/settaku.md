# 대탁 (セット / setto)

| 항목 | 내용 |
|------|------|
| 일본어 | セット |
| 로마자 | setto |
| 한국어 | 대탁 |

## 정의

자신 포함 4명을 미리 모집한 후, 마장에서 탁자 1개를 빌려서 해당 멤버끼리 치는 행위. 일본에서는 '세트'라는 표현을 쓴다.
